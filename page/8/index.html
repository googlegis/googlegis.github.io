独自带娃去旅游-3

发表于 2014-06-02 | 分类于 育儿经 |

6.02

今天准备回去了，6.自然醒。

收拾东西，烧水，洗脸，等娃子醒。

娃子没有睡懒觉，6.30就醒了。

穿衣服，洗脸，喝奶。准备出发。

去服务台办理退房手续，两夜加上一瓶天价水共485，看来每夜才235，真的是相当便宜。

同学真是太给力了。

10分钟办好手续，走人。娃子抱着她的大熊氢气球，我抱着娃子，背着双肩包。Let’s Go!

到地铁站买票，进站时，氢气球不让带进站，放气收起来让娃子拿着。

地铁里，娃子明显疲软，想晕车，就抱着她，让她睡觉。

中南路倒车，地铁出站娃子还在睡觉呢。

在火车站前的石球上坐着歇着，娃子睁开眼：爸爸，好多人啊，这里有人，这里也有人。

娃子一个最大的特点，就是睡醒不迷糊。

带娃子进站，找候车室，一切顺利。

等车间隙，娃子又是各种调皮，爬高上低，左顾右盼。

怕她在车上晕车，总是逗她玩。

到点上车，还好是最近的车道，不用爬楼梯。

整好走人。

在车上，先是找到座位，开始不老实的乱坐。新鲜了半个小时，然后开始想闹人，我赶紧搂住她哄她睡

觉，怕她又晕车。一会睡着，一直到合肥才醒。差不多一半路程了，下面就容易对付点了。

只要不晕车，其他的都不是事。

醒了，先给她找点饼干吃，然后带她去厕所尿尿，洗脸回来喝了点水，去餐车找饭吃。

到地方才发现，没有餐车，只有卖盒饭的，要了个奥尔良烤翅的盒饭。

娃子喜欢吃烤翅上的皮。四块烤翅吃了三个的皮，最后一个掉地下了。

我吃了一份饭。

然后和娃子各种耍。

3.30左右，娃子坚持不住了，闹人，又是一阵哄。

4.08分，列车准点到达上海虹桥，带娃子下车出站，在门口等妈妈。

妈妈不知道上哪里去了，给妈妈打电话，娃子闹着我要打，我要打，

本来就着急没有让她打，娃子伤心的哭了。

等妈妈来，也不理妈妈，然后妈妈用樱桃把娃子哄的不哭了。

娃子连连吃了很多樱桃，然后不哭了。准备坐地铁回家。

可能因为和6.4连的太紧了，地铁入站口，查得相当严。门口排起了长队。

进去之后，找了个车头位置人少的地方恰好能找到座位。

路上娃子可能有晕车反应了，搂着她睡觉了。

到小区门口娃子醒了，电梯上楼，到家门口，娃子不愿意了，哭着闹着：上爸爸上学的地方吧！

老婆和丈母娘说她不想家，我想可能是因为3天之内，时空变换太快，她还没有适应一个地方就立刻

换了，所以一下子没有接受。抱着在门口哄了20分钟，终于愿意进屋了，进屋之后，黏在我身上不

下来，又是一阵好哄，最后妈妈用樱桃把她哄下来了。

10分钟后，娃子又恢复到3天前的状态了，到处乱跑乱蹦。O(∩_∩)O哈哈~，调皮的小娃子又回来了。

独自带娃去旅游-2

发表于 2014-06-01 | 分类于 育儿经 |

6.01

今天是儿童节，在酒店睡到自然醒，其实也不算。

因为自从有了娃子之后，我总是6.左右就自然醒。

起来先烧上水，然后发现免费的水昨天晚上喝完了。

没有计算好，只有从小冰柜了拿了已个天价矿泉水，15元RMB。

忍了，娃子还在睡，不能自己出去买，同学们昨天晚上吃完饭又去唱k，估计凌晨2.才睡吧。

收拾背包里的东西，然后把自己的衣服洗了，又想起来刮胡刀没有带。

娃子7.的时候，自己爬起来：爸爸，俏俏要喝奶。

赶紧整水弄奶，抱着娃子在窗户那里看着大车喝。

窗户外面跑着越来越多的双层巴士，说明外面的人开始活动了。

喝过奶之后，娃子竟然又睡着了。睡吧。

趁娃子睡着，打开电视看一会，怕娃子睡得太多，晚上又不睡。看会电视也没有事。

外面下着雨，还不知道等会同学们要去哪里。

8.娃子醒了，烧好热水，给娃子带了一个雨衣，拿着酒店提供的雨伞，带娃子出去觅食。

出酒店往学校方向，不到红绿灯，有个 郭记肉夹馍 ，进去要了个肉夹馍，要了一碗红豆粥，

又看有粽子，想着明天端午节呢，回到家不知道几点，先要两个粽子尝尝，全程娃子都很配合。

自己吃饭，自己吃粽子，吃肉夹馍。中间碰到皇兄携夫人一起来就餐。

吃好出去，娃子就不是那样了，要爸爸抱着，东西也不让别人帮拿。然后就一路抱到学校门口。

学校门口几个同学正在玩，然后他们要去吃饭，我就不去了，我和娃子刚吃过，就抱着娃子去

学校里面。还是那个小门，顺着熟悉的路往前走，可以看到前面的南湖，到以前的篮球场那里，

现在改成了体育馆，顺着斜坡上去，就来到以前住的2栋。把娃子放在门前的雪松下，照了几张照片，

然后娃子就往前走，我喊她，她反而不理我，刚下过雨，她踩着积水就走过去，鞋子什么的都湿了。

我过去一把抓住她，她就委屈的哭了： 爸爸，俏俏要走。

“那你的鞋子湿了没”

她不管，“爸爸，俏俏要走！”

在宿舍前面，一个女生喊道：别哭！

我突然意识到，我错了，我本来可以用其他的东西把她哄过来的，为什么针对她进行教育呢。

然后抱着她：爸爸抱你去看鱼鱼好不好，那边水里有很多鱼。

哄了半天不哭了。

在南湖边上绕了一圈，在操场绕了一圈，她又想踩水，没让，又委屈了一小会。

在主席台躲雨，同学们打电话过来说去认领的树那里照相，就过去。

同学们已经在那里了，有个女生还送给娃子一个氢气球，大熊的那种，小娃娃特喜欢，可是她不愿

叫人，也不愿意别人摸她，就在那里喊着：爸爸，我们回去吧，爸爸，我们回去吧。

终于在一个男生摸了她一下之后，彻底的发作了，哭得稀里哗啦的。

没有办法，不能和同学去玩了，抱娃子回酒店睡觉。

11.40左右了，娃子该睡觉了。

一路上，娃子就是不下来，一直都是让抱着，把她放下来，她就耍赖。

今天是6.1，儿童节，她的节日，她最大，就抱着吧。不就是胳膊累了点吗。

走到一个良品铺子，买了一点她喜欢吃的东西。这下抱着她，我就更累了。

刚才过来的时候，没有感觉那么远啊。胳膊累的又酸又疼，几次把她放下来想歇会，她就把腿蜷起来，

脚尖歪着，不让放到地下。没有办法，只有硬撑着往前走。

还好，我之前一个星期，每天都跑5公里，锻炼了足够的脚力和耐力，就是胳膊累的很，臂弯那里酸

疼的厉害。

谢天谢地，终于到酒店，喝水，休息。然后，然后娃子就来精神了，带着娃子玩，玩了差不多两个小时，

娃子终于睡着了，我也可以休息睡觉了。同学们到5点左右才回来。

6.左右，娃子醒了，整点水喝，休整一下，出去觅食。

来到早晨的那个肉夹馍店，要了一碗牛肉面和一碗鸡蛋面，我知道娃子肯定饿了，又找了个小碗，给她

一双筷子，挑一小部分放到她小碗里，娃子吃的津津有味。吃了还吃，吃了还吃，足足吃了有5小碗。

这下吃饱了，还惊奇的发现，娃子竟然会用筷子了，以前都是孙悟空吃面条，两只筷子手心里攥，这次

竟然知道用拇指和食指夹住，然后用无名指和小指去挑动筷子往嘴里扒。于是夸她会用筷子了，她吃的

更欢了，还骄傲的对我说：爸爸，你可知道为啥吧，俏俏长大了。

吃完饭，回酒店，同学们休息的差不多了，他们相约去步行街，去吃烤肉，我抱娃子去7层找他们玩，

小娃子见了他们又是一个一不理二不叫，一个阿姨用萨琪玛和饼干才换来她的直视，仅仅是直视。就是不

和人家说话。

过了一会他们该走了，我不能去了，娃子估计9.30左右又该闹着睡觉了，我到那里也不能玩，抱着还那

么重。

回到房间继续和娃子玩，上午有个狗熊的球球，下午有饼干，还有自己的芭比娃娃。

同学怕我吃不上饭，还给我送来一份白粥，真的是万分感谢。

9.30时，娃子再一次安静的睡着了。

这一天，真的是最累的一天，胳膊肘子快要累断了。

躺下歇歇。喝点水，看会电视，看看昨天的衣服和今天的衣服干了没，翻翻晾晾。希望明天走的时候不湿

就好了。

把娃子的玩具和衣服收拾好，然后看了一会电视睡着了，同学发消息来邀我去玩，要给我带烤肉回来吃，

好意心领了，太晚太困了。睡觉了先，明天要早起赶地铁去火车站。

独自带娃去旅游-1

发表于 2014-06-01 | 分类于 育儿经 |

这事还得从头说起，那一天，有同学给沉寂的校友群里扔了个炸弹：毕业十年，要不要回母校聚一下。

母校，武汉中南民族大学，我们进去的时候，它还叫中南民族学院，我在上学期间简直是最沉闷的一个，

不交友，不学习，不上课，不社团，就是一个人在宿舍那个集体里还能说上几句话的人。记得毕业的时

候，有人问我：你是不是对我有意见，你从来都没有和我说过一句话! 我尴尬的笑笑，又没有说话，走开

了。不好意思，我和1/3的同学都没有说过一句话。

十年了，从毕业时的寸步难行，到中间的重创受伤，到现在温暖的小窝，乖巧的小娃子，我满足了。

说来就来，宿舍老大打电话过来询问能够参加，第二天回复，可以参加。还有个决定是带着娃子去。

女儿27个月了，脾气暴躁，爱闹人，不过很精明，比我小时候聪明多了，总是黏着我，爸爸爸爸的跟在

屁股后面。独自带娃出去，我也是想证明给老婆看，我一个人也能把娃子带好。

提前一周，开始给娃子上教育课，再过一周，到星期六，爸爸带你去坐大火车，特别大，你要听话好不

好。

娃子总是特别希翼的眼神望着我，狠狠的：嗯！

5.31

早晨老婆和丈母娘早早的5.30分就出去坐车游玩了，就剩下我和娃了，我起来给娃穿好喝奶，检查

收拾东西，为了路上方便，把所有衣物都装在一个包里，另外再抱个娃，这样就容易一些了。

7.30分开始离家出走，门口打的，虽然知道娃子可能会晕车，但是想到地铁里冷冷的空气，和漫长的转

车等车时间还是决定了打车去，这一决定，直接导致了娃子后面一整天的呕吐不止。

一路上，娃子是晕头转向，我搂着她，让她睡觉，娃子则是迷迷糊糊的一路睡。

大概8.30分得时候到达虹桥火车站，我动了动准备下车，娃子睁开眼问：爸爸，好了没有啊。

我说到了，师傅找了钱，拿包下车，两步刚走，娃子哇的一声全吐出来了。

衣服上全部都是，没有办法，把娃子放下来，从包里拿出干净衣服换上，把脏衣服用塑料袋装好放包里，

进站换票，换票点简直是人山人海，小娃子受不了那个味道，一直在问：爸爸，好了没有啊？

听着都心疼。

终于把票换出来，带娃上去找东西吃，

肯德基只有汉堡套餐，而且品种很少，去了旁边的永和，要了一份套餐：皮蛋瘦肉粥和蛋饼油条。

小娃子特别喜欢喝粥，忘记带小碗和勺子，就用盛蛋饼的盘子给娃子冷粥，然后顺着盘子娃子喝的又香

又甜。

第一次带娃出去，心里紧张，娃子喝的衣服上都是米粥，娃娃说：我要和妈妈说话，妈妈，我喝粥弄到

衣服上了，你快点回来吧，俏俏想你了！

泪水差点夺眶而出。

开弓没有回头箭，只有自己加油了。

吃完饭，我候车室等车，娃子各种调皮，一会要站凳子，一会要跑到那边，一会要喝凉水，左哄右挡，

终于到车上了，还是没有经验，自己选了一个靠窗户的三人座最里面的座位，为后来的照顾小娃子添加了

很大的的阻力。

娃子先上来很兴奋，爬高上低，左顾右盼，慢慢的就开始疲软了，慢慢显示出晕车的症状来。

没有精神，想闹人，软绵绵的趴在我怀里一动不动，还不想睡觉，左哄右哄，终于睡着了，也许是因为火

车晃动如摇篮的原因，娃子睡得很香，一直睡到一半路才醒。

然后才是艰难的时刻。

醒了之后，带娃子去尿尿，然后洗脸回来，想起来妈妈给带了一个士力架，拿出来给娃子吃，娃子自然

是不讲意思，大口吃。一个人吃了将近一半，然后又闹着喝了一点凉的矿泉水，看着娃子大吃二喝，心

想，应该没有问题吧，吃就吃吧。

半小时之后，娃子开始有反应了，又全身疲软下来，躺在我怀里一动不动，闹人似的吭唧。

过了十分钟，突然( ⊙o⊙ )哇的一下全吐出来了。吐的是稀里哗啦，眼泪汪汪。没有办法，只好搂了又

搂，哄了又哄，带娃子去洗手洗脸，还好娃子身上吐的不多，我的裤子上倒是不少，我也不换了，赶紧

哄娃。

娃子从此开始吐的历程，一直状态不好，在怀里吭吭唧唧的闹人，不躺不做，就让抱着，还得站起来出

去抱着，没有办法，抱着她在列车接头处走啊走，晃啊晃，想把娃娃晃睡着，可能是晕得太厉害了，娃

子一直都没有睡着，抱着抱着，娃子又哇的一下全吐了，这次吐的鼻子里都冒水了，只能抱着娃赶快哄，

怕她自己害怕。

娃子精神稍微好点了，要喝水，又把带的热水给她喝，喝了好多水，我想水总没有事吧。

半小时之后，事实推翻了我的想法，娃子连清水都吐出来了。娃子在怀里喊：爸爸，咱们下去吧。

没有办法，宝贝，爸爸下次在家和你玩，也不带你出来了。

好歹下车了，收拾东西，抱着软趴在肩膀的娃娃下车，又给她换了一套衣服。

走，出去坐地铁去。

地铁倒是好找，买票，坐车，车上真冷。嗖嗖的风，用双臂护着娃娃的胳膊，怕娃娃再一受凉，发烧了就

麻烦了。

出地铁10分钟，到酒店，check-in 安排好，给娃子整点热水，洗洗脸，然后放水洗衣服。

同学们说晚7.在饭店集合吃饭，已经6.20了，而且还不知道饭店在哪里。整到6.40，同学打电话过来

问在哪里，稍微收拾了一下，准备出发，发现剃须刀没有带，前天又没有刮胡子。唉， 凑合吧。

路上碰到同学回来拿东西，于是一起去饭店。 老师，同学都已经到了，嘻嘻哈哈的说不停，十年没有

见了，加上上学的四年，总共有14年了，我住的酒店在14楼，上海租得房子也在14楼。真巧。

段老师还是那样温文尔雅，特漂亮;司老师还是那样说话声音响亮，声线往上飘；后面来的韦老师也是

老样子，一点都没有怎么变样，再看同学们，男的成熟稳重了，女的纤瘦漂亮了，大家都在自己的路

上稳步前行，还是带着10年前的迷人的微笑，一点都没有变，真好。

两桌同学，20几个人，见了面还是那样亲切自然。还是那样亲热招呼。真好。

大家坐下，领导讲话，老师献词，回忆了一下过去，表达了一下现在，畅想了一下未来。

然后就开始酒过三巡，不分大小了，走过来串过去，大家开始互相把酒言欢了。彼此高声畅谈，

真的是酒不醉人人自醉，抽刀断水水更流。

小娃子在老师献词的时候就开始闹着要吃东西。我又不好直接起来拿东西吃，就和她谈条件：乖一点，

等会爸爸给你拿黄瓜。

在Mac下编译Android源码

发表于 2014-05-15 | 分类于 Android |

最好的教程就是官方文档。

1.需要至少15G大的大小写敏感分区，以免build debug版本空间不够

在Mac下编译android的源代码需要一个大小写敏感(Case Sensitive)的分区格式如果你原来的分区不是HFS一类大小写敏感的文件格式，那么最简单的办法就是新建一个这种分区格式的dmg出来，命令如下：

sudo hdiutil create -size 15g -type UDIF -fs HFSX -volname android ~/android.dmg
sudo chown $USER ~/android.dmg

1)双击android.dmg，mount volume到/Volumes/android。

2)进入系统偏好设置–>帐户–>登录项–>点”+”，选android.dmg这样系统启动的时候会自动mount volume

2.下载代码或者直接拷贝已有源代码到/Volumes/android目录下

下载源码需要安装Repo工具，该工具是为了更好的使用Git源代码管理软件（类似于CVS等版本控制系统），执行以下命令创建~/bin目录：

$ cd ~
$ mkdir bin
$ echo $PATH

下载Repo：

$ curl http://git-repo.googlecode.com/files/repo-1.13 > ~/bin/repo
$ chmod a+x ~/bin/repo

之后在/Volumes/android目录下创建android目录以存放android源码：

$ cd /Volumes/android
$ mkdir android
$ cd android

接着初始化Repo客户端：

$ r~bin/repo init -u https://android.googlesource.com/platform/manifest -b master

成功后显示：

repo initialized in /Volumes/android/android

这时在/Volumes/android/android目录下就会有.repo的目录用来同步manifest文件，该文件是用来同步源码的，下载源码使用命令：

$ repo sync

大概要下载4G多的文件，所有下载过程会比较长。

非常重要的，下载完成之后需要打几个补丁，请分别执行以下命令，按顺序打以下四个补丁：
cd build
git pull git://android.git.kernel.org/platform/build refs/changes/74/12074/3

cd system/core
git pull git://android.git.kernel.org/platform/system/core refs/changes/45/11845/3

cd external/qemu
git pull git://android.git.kernel.org/platform/external/qemu refs/changes/46/11846/3

cd system/core
git pull git://android.git.kernel.org/platform/system/core refs/changes/97/12597/1

3.下载并安装XCode和MacPorts

首先下载安装最新的XCode，下载地址：http://developer.apple.com
然后下载MacPorts，下载地址： http://www.macports.org/
修改~/.profile文件，添加：

set the number of open files to be 1024

ulimit -S -n 1024

4.编译源码

首先修改~/.profile文件，内容类似：

MacPorts Installer addition on 2010-05-19_at_19:14:53: adding an appropriate PATH variable for use with MacPorts.

export ANDROID_PRODUCT_OUT=/Volumes/android/Android-Source/out/target/product/generic

export ANDROID_PRODUCT_OUT_BIN=/Volumes/android/Android-Source/out/host/darwin-x86/bin

export PATH=/opt/local/bin:/opt/local/sbin:${ANDROID_PRODUCT_OUT_BIN}:${ANDROID_PRODUCT_OUT}:$PATH

Finished adapting your PATH environment variable for use with MacPorts.

MacPorts Installer addition on 2010-05-19_at_19:14:53: adding an appropriate MANPATH variable for use with MacPorts.

export MANPATH=/opt/local/share/man:$MANPATH

Finished adapting your MANPATH environment variable for use with MacPorts.

然后打开一个新的命令行窗口，执行以下命令：

$ cd /Volumes/android/Android-Source
$ make

源码编译时间较长，大概需要4～5个小时，请耐心等待，编译完成之后，执行命令：

$ emulator

将 调出模拟器，说明编译成功！
