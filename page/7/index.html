他们都比我有钱

发表于 2014-07-05 | 分类于 爱就趁现在 |

本来以为自己的努力已经够了，谁知道连底层都不算，最多也就是个乞丐级别的人物。

突然之间自己就没有钱了。如同自己像狗一样的流浪，哪里有一点像是有家有室的样子。

据目前的局势来看，他们都比我有钱，我不是嫉妒他们，而是恨我自己。通过过

来，竟然以自己没有钱的原因而躲开了。:-(

这是TMD的什么精神，这是安逸享乐不思进取的精神。

整天的老婆孩子热炕头，你侬我侬大家侬，没有钱凭什么给老婆美丽的外表，

没有钱凭什么给孩子美好的未来，没有钱凭什么给父母安享晚年。

总是以为自己努力够了，每天假装忙碌的样子，好让自己看起来很踏实的样子，

可是钱不到手，一切都是白搭。

钱，谁能和钱过不去。

钱根本就是省下来的，钱是挣出来的。

凭什么别人半夜两点睡，你10.半就能爬上床，凭什么别人在那里拼死拼活的加

班，你却抱着娃在那里悠哉悠哉的逛大街。什么是差距，这就是差距，

差距就在于别人能买房开车，而自己却只能租房公交；别人35岁之前已经能够

安定的生活，自己35岁还在像刚毕业的毛头小伙子一样毛手毛脚。

朝自己脸上扇一耳巴子，告诉自己，该醒醒了。

chrome voice search has been turned off

发表于 2014-07-05 | 分类于 Mac |

Fix Chrome’s Voice Search has been turned off’

From Chrome menu select Settings, select Show Advanced Settings

Privacy>Content Settings scroll down to ‘Media’ make sure ‘ask me when a site requires access to my camera and microphone (recommended)’ option is selected (check the screenshot below) instead of this one ‘Do not allow sites to access my camera and microphone.’

Click ‘Manage Exceptions’ and remove any exceptions you might’ve added for Google site in past which showing Deny words for Audio and click ‘Done’ button to save the changes.

Advanced Settings Media thumb Fix Chrome is Showing Voice search has been turned off. Details Error

Now try to start voice search in Chrome and it works as expected without showing any errors. FYI, ‘Ok Google’ has came to desktop Chrome, you need to install Google Voice Search Hotword beta extension for that. After installing that extension you can talk to Google without clicking on microphone icon or you just need to say ‘ Ok Google’ then on Google homepage.

独自带娃去旅游-3

发表于 2014-06-02 | 分类于 育儿经 |

6.02

今天准备回去了，6.自然醒。

收拾东西，烧水，洗脸，等娃子醒。

娃子没有睡懒觉，6.30就醒了。

穿衣服，洗脸，喝奶。准备出发。

去服务台办理退房手续，两夜加上一瓶天价水共485，看来每夜才235，真的是相当便宜。

同学真是太给力了。

10分钟办好手续，走人。娃子抱着她的大熊氢气球，我抱着娃子，背着双肩包。Let’s Go!

到地铁站买票，进站时，氢气球不让带进站，放气收起来让娃子拿着。

地铁里，娃子明显疲软，想晕车，就抱着她，让她睡觉。

中南路倒车，地铁出站娃子还在睡觉呢。

在火车站前的石球上坐着歇着，娃子睁开眼：爸爸，好多人啊，这里有人，这里也有人。

娃子一个最大的特点，就是睡醒不迷糊。

带娃子进站，找候车室，一切顺利。

等车间隙，娃子又是各种调皮，爬高上低，左顾右盼。

怕她在车上晕车，总是逗她玩。

到点上车，还好是最近的车道，不用爬楼梯。

整好走人。

在车上，先是找到座位，开始不老实的乱坐。新鲜了半个小时，然后开始想闹人，我赶紧搂住她哄她睡

觉，怕她又晕车。一会睡着，一直到合肥才醒。差不多一半路程了，下面就容易对付点了。

只要不晕车，其他的都不是事。

最好的教程就是官方文档。

1.需要至少15G大的大小写敏感分区，以免build debug版本空间不够

在Mac下编译android的源代码需要一个大小写敏感(Case Sensitive)的分区格式如果你原来的分区不是HFS一类大小写敏感的文件格式，那么最简单的办法就是新建一个这种分区格式的dmg出来，命令如下：

sudo hdiutil create -size 15g -type UDIF -fs HFSX -volname android ~/android.dmg
sudo chown $USER ~/android.dmg

1)双击android.dmg，mount volume到/Volumes/android。

2.下载代码或者直接拷贝已有源代码到/Volumes/android目录下

下载源码需要安装Repo工具，该工具是为了更好的使用Git源代码管理软件（类似于CVS等版本控制系统），执行以下命令创建~/bin目录：
