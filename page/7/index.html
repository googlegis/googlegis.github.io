世界杯中的德国队

发表于 2014-07-09 | 分类于 文章 |

今早老婆问我，最近有没有给世界杯写点文章，我说还没有

老婆说还是写点吧，毕竟你还是看球的。

凌晨的世界杯 德国-巴西 7:1 .

谁能想到，世界上还有人敢菜巴西7个球。

其实对德国队偏爱有加，就算他们在没落的时期，98年的时候还不怎么看球，只是

听闻有足球世界杯这回事，知道那个时候有个贝克汉姆，知道德国的三驾马车已经

没落。然后就是整个高中时期。

后来上大学，正好赶上2002年的韩日世界杯，热血青年加上平时也和同学踢球，就

看球看了个天昏地暗，那个时候的德国队应该是属于巴拉克和卡恩的年代。没落了。

但是也就是从卡恩和巴拉克身上开始喜欢德国队，坚韧刚强。

2006年正好赶上事业的上升期，把世界杯这茬给忘了，翻资料来看，那个时候，

拉姆、波多尔斯基、博罗夫斯基、施奈德等已经在和巴拉克/卡恩那个阶段的人开始

交替了。

2010年世界杯，就会发现，德国队的阵容完全变了，这里面只有克洛泽是老人了，

但是他永远都是一个年轻的面容，诺伊尔/拉姆/施魏因斯泰格/厄齐尔/戈麦斯/穆勒，

这些现在牛逼的人2010年就出现在世界杯上，那时的我也在想，这不是娃娃军团吗？

这来打什么球啊？明知他们不会走很远，但是还是默默的注视着他们，等4年之后吧。

4年之后这些人就牛逼了。

2014年世界杯，上届世界杯上的那些人又回来了。

踢巴西 7:1, 看着巴西球迷的哭泣，真的感到可怜，可是足球就是战场，没有人会怜悯

你，就像当年没有人会怜悯德国队一样。

联想到前面已经走的那几个球队，西班牙/英格兰/意大利/法国，有的还是老弱病残占

据主力不肯下来，有的则是缺乏血液循环，后续跟不上，再则是新手还没有大赛经验

需要时间和经历来磨练。

这么看来，后面无论是阿根廷或者荷兰，还真的不是德国的个了。

好吧。德国队，这次你牛逼起来吧。也不枉我支持你12年了。

已经走了的那几个国家队，回去更换血液，积累经验，下次再来吧。

巴西，这个国家不想说，从小罗不踢的那个时候就开始走下坡路，足球王国也毕竟是

皇帝的新装，缺一两个人整个球队就溃不成军，德国队则是那个说实话的小孩。

足球只会用进球来说话。

足球，两个字，进对方的球门，对方进自己的球门。

进对方球门的队伍留下，进自己球门的队伍出去。

chrome voice search has been turned off

发表于 2014-07-05 | 分类于 Mac |

Fix Chrome’s Voice Search has been turned off’

From Chrome menu select Settings, select Show Advanced Settings

Privacy>Content Settings scroll down to ‘Media’ make sure ‘ask me when a site requires access to my camera and microphone (recommended)’ option is selected (check the screenshot below) instead of this one ‘Do not allow sites to access my camera and microphone.’

Click ‘Manage Exceptions’ and remove any exceptions you might’ve added for Google site in past which showing Deny words for Audio and click ‘Done’ button to save the changes.

Advanced Settings Media thumb Fix Chrome is Showing Voice search has been turned off. Details Error

Now try to start voice search in Chrome and it works as expected without showing any errors. FYI, ‘Ok Google’ has came to desktop Chrome, you need to install Google Voice Search Hotword beta extension for that. After installing that extension you can talk to Google without clicking on microphone icon or you just need to say ‘ Ok Google’ then on Google homepage.

是不讲意思，大口吃。一个人吃了将近一半，然后又闹着喝了一点凉的矿泉水，看着娃子大吃二喝，心

想，应该没有问题吧，吃就吃吧。

半小时之后，娃子开始有反应了，又全身疲软下来，躺在我怀里一动不动，闹人似的吭唧。

搂，哄了又哄，带娃子去洗手洗脸，还好娃子身上吐的不多，我的裤子上倒是不少，我也不换了，赶紧

哄娃。

娃子从此开始吐的历程，一直状态不好，在怀里吭吭唧唧的闹人，不躺不做，就让抱着，还得站起来出

去抱着，没有办法，抱着她在列车接头处走啊走，晃啊晃，想把娃娃晃睡着，可能是晕得太厉害了，娃

子一直都没有睡着，抱着抱着，娃子又哇的一下全吐了，这次吐的鼻子里都冒水了，只能抱着娃赶快哄，

怕她自己害怕。

娃子精神稍微好点了，要喝水，又把带的热水给她喝，喝了好多水，我想水总没有事吧。

半小时之后，事实推翻了我的想法，娃子连清水都吐出来了。娃子在怀里喊：爸爸，咱们下去吧。

没有办法，宝贝，爸爸下次在家和你玩，也不带你出来了。

好歹下车了，收拾东西，抱着软趴在肩膀的娃娃下车，又给她换了一套衣服。

走，出去坐地铁去。

地铁倒是好找，买票，坐车，车上真冷。嗖嗖的风，用双臂护着娃娃的胳膊，怕娃娃再一受凉，发烧了就

麻烦了。

同学们说晚7.在饭店集合吃饭，已经6.20了，而且还不知道饭店在哪里。整到6.40，同学打电话过来

问在哪里，稍微收拾了一下，准备出发，发现剃须刀没有带，前天又没有刮胡子。唉， 凑合吧。

路上碰到同学回来拿东西，于是一起去饭店。 老师，同学都已经到了，嘻嘻哈哈的说不停，十年没有

见了，加上上学的四年，总共有14年了，我住的酒店在14楼，上海租得房子也在14楼。真巧。

段老师还是那样温文尔雅，特漂亮;司老师还是那样说话声音响亮，声线往上飘；后面来的韦老师也是

上稳步前行，还是带着10年前的迷人的微笑，一点都没有变，真好。

两桌同学，20几个人，见了面还是那样亲切自然。还是那样亲热招呼。真好。

大家坐下，领导讲话，老师献词，回忆了一下过去，表达了一下现在，畅想了一下未来。

然后就开始酒过三巡，不分大小了，走过来串过去，大家开始互相把酒言欢了。彼此高声畅谈，

真的是酒不醉人人自醉，抽刀断水水更流。

小娃子在老师献词的时候就开始闹着要吃东西。我又不好直接起来拿东西吃，就和她谈条件：乖一点，

等会爸爸给你拿黄瓜。

没有想到这家伙得寸进尺，大声喊：要吃黄瓜。

走远，附近的人就举举杯，说上两句，大家都还好，来回走动着，互相喝酒说话。

没有办法，抱她去厕所尿尿，然后洗洗手和脸，上屋里找个同学让代说一下，抱娃子回酒店了。

路上娃子也没有睡，到酒店来精神了。可能是因为在火车上一直睡，现在不困了，就陪着她一直玩，

要给她洗澡也不让，就另外再换了一个t恤，准备睡觉穿。

看电视，穿着袜子在地毯上跑，要喝水，要看画书，要玩芭比娃娃，终于玩到11.30 睡着了。

这一天真累。洗澡睡觉。
