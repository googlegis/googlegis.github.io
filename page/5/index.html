在 Mac 上安装 Xamarin

发表于 2016-07-05 | 分类于 Xamarin |

最近因为工作的原因，要把已有的 .Net Web 项目进行跨平台开发研究，项目部署

在 CentOS 上，正好有机会研究一下 Xamarin 的开发，所以决定把这一系列

的研究过程都记录下来，备忘和分享！

.Net 的跨平台开发，现在无非是两种方式， 一种是 Mono SDK, 一种是

Asp.Net Core. 但是 Asp.Net Core 还是太新，只能作为对比，还不敢拿

来直接作为开发使用。所以，默认使用 Mono。

Mono 是一个跨平台的 .Net 解决方案，最后被微软收购并进行维护和支持。

应该说还是有一定的群众基础的。

在 Linux 系统下推荐的是 MonoDeveloper

http://www.monodevelop.com

在 Windows 和 Mac 系统下推荐的是 Xamarin Studio。

http://www.monodevelop.com/download/

从 Mono-Develop 的官方页面下载的版本是 5.6

但是从 Xamarin 的官方页面下载的是 6.0

https://store.xamarin.com/account/my/subscription/downloads

如果只是开发 Web 页面，只下载 Xamarin Studio 的安装包即可。

然后直接双击安装， 如果没有安装 Mono SDK，会提示要先安装 SDK。当然也可以

从网站先下载安装 Mono SDK，安装再安装 Xamarin Studio。

http://www.mono-project.com/download/

打开 Xamarin Studio，新建 Asp.Net Mvc 项目进行测试。

至此，说明 Xamarin Studio 安装成功！

加载后，打开 readme，里面有个卸载的命令，先运行一下。

如果命令行中仍然显示旧的版本号，不要着急，只是程序的路径没有添加到环境变量中。

打开命令行工具：

jays-MacBook-Pro:~ jay$ vim ~/.bash_profile

把下面这行添加到 .bash_profile 文件的后面，

export PATH=/usr/local/git/bin:$PATH

保存文件，退出编辑！

再验证版本，就会看到最新的升级版本号。

mac 到底值不值这么贵

发表于 2015-12-11 | 分类于 文章 |
题图 操作系统全集 前-MacOS 后右-Windows10 后左-Linux(Ubuntu)

2014年春，买了RMBP 2013 later，在32岁的年龄，还是冲动了一下，做一次自己喜欢的事情。

首先第一条，mac的笔记本到底贵不贵？

贵不贵看情况，如果你都是在windows下的开发，而且对mac的开发一点兴趣没有，买个mac就是为了装13的，那mac真贵，贵到你连看都不想看。随便一个本子就MBA也的7000+，就不说联想了，dell和hp的机器也就3000-4000配置就很好了嘛。

然后再看配置，首先一个ssd的硬盘，512G去jd上搜一下，看看多少钱，再看看微软的surface，512G的ssd配置多少钱? 然后再看mac是不是贵，是不是值。

第二条，mac上面什么都不能做啊？

那么多exe的软件在mac下鸟用没有，你让我怎么办？我想装个网银都不能用，还有360的杀毒也不能用，那要是中毒了怎么办？亲，选对的，别选贵的。

第三条，mac下我能做什么呢？

对于普通用户来说，真的什么都做不了，就算打开个word还要转换成pages下面才能编辑，而且保存后的文档别人打不开，需要转成word别人才能用，这不是花钱找罪受吗？
如果你确实生活在windows的圈里出不来，还是不要出来了。

第四条，mac的用途:

1. 连接iPhone，牛叉。

你会觉得以前在windows下载安装iTunes，就像是带套去ML，虽然也是嗯嗯呀呀的费半天劲，但是总是感觉少了一点什么？到底是什么呢？少了一点速度和激情。

2. 超高的分辨率，

这个只是针对我以前的那个旧pc来说的，换了之后，感觉整个世界都清晰了，连上楼都有劲了。

3.完整的办公及娱乐套件

iWork 和 iLife 的应用，够你用的了。而且现在买的mac里这些应用都是免费的。对了，不说到想不起，你的windows下的office 是正版的吗？

4. 免费的系统

这个我还真不知道从哪个osx的版本开始的，我开始用的时候就10.9 了。反正一直都是免费，这个对windows来说，简直就是天大的好事，连升级都要收费，吃多了吧。不过windows最近也开始学乖了，推出了一系列免费及优惠政策。

5. 商店 iTunes Store 及 APP Store

以商店模式进行应用的索引及排名，提供开发机会。
这样的模式对开发者来说再好不过，只要你有创意，有技术，还是可以站着把钱挣过来的。(国内的那些歪门邪道暂且不表)

6. 牛X的Photos

为什么单独把它提出来说，在买这个mac之前，我在windows下还真没有看到哪个应用或者程序能够做的如此友好而且精巧，不得不表彰一下。

第五条，比如其他的基于linux、安全、性能优化、wifi共享…… 可惜我不是技术流，对这些都是知，而不解。只能提提而已了。

对任何事情都有个先入为主的概念，比如对初恋，比如对第一部手机，比如对windows，如果你刚开始接触电脑时，就开始使用macOS系统，而不是windows，恐怕现在是另外一种想法了。

所以换不换，用不用，全在你一念之间。

–完–

也许斯嘉丽和肿瘤男约好了时间和地点，当斯嘉丽没有等到肿瘤男的时候，她知道，事情穿帮了。

她毅然决然的独自一人沿着公路走了。

然后她尝试吃人类的食物，然后吐掉。她尝试坐人类的公交，她尝试听人类音乐，跟着节奏弹手指，

她尝试爱一个人和被一个人爱，她尝试和男人做爱，但是她没有成功，在那个对他温柔体贴的男人尝试

了N次没有成功之后，她连忙跑起来，用灯照着自己的阴部。

也许她只是人造人，身体结构还没有完全和人类一样。

一直到最后，在森林里遇到伐木工，被伐木工性侵未遂，暴露身份后被伐木工用汽油烧死之后，

在茫茫雪地上只留下一股黑烟，影片结束。

但是我的心里却久久不能平静，剧本太牛，导演太牛，演员也太牛了。

当一个杀人工具，一个只负责引诱人类到黑色泥潭送死的人造人，在发现人类的感情和生活后，

想认认真真的做一个人的时候，却被正是她所希望成为的人类性侵和杀害。

人类中有好人，也有坏人，人性中有善良的部分也有丑陋的部分。只是这些，斯嘉丽还没有学会。

她以前接触的虽说有点淫欲但还不致于害人的人类，尤其是那个为救人在海边累晕，然后被斯嘉丽敲死，

在他嗷嗷大哭的小孩面前拖走的男人。肯定对斯嘉丽的心灵也有触动。

斯嘉丽开始引诱别人时，总是问住哪里，做什么，到哪里去，家里都有谁。

我想她其实不知道这些话代表什么意思，只是按照固定的模式说出来。

但是这些形形色色的人的生活对她也有很大的影响。原来人类可以有这么多姿多彩的生活。

她在温柔男的家里欣赏自己的躯体的时候，才第一次真真正正的把自己当做一个真正的人来看。

她把肿瘤男引诱后在门后面看镜子里的自己的时候，我就觉得有蹊跷。果然肿瘤男被她放了。

也许从那开始，她决定开始考虑人类的生活方式了，在肿瘤男失约后，她决定离开这个地方，自己往前走。

她该是多人想成为人类啊。被伐木工撕烂皮之后，一个人把头拿下来的时候，眼睛里闪烁着多么的不舍和哀怨。

在被伐木工浇上汽油烧起火逃跑的时候，手里抱着的依然是不舍的人类的头颅。也许是不愿意，也许是不

甘心，也许是想在生命的最后一刻多抚摸一下自己的脸庞。

可是所有到最后的一切都只不过是一缕黑烟。

她想成为的人类把她杀了。

也许她只是一个皮囊的黑炭人，她没有人类的思想和感情，可是她在努力去学，去感受，去思考，

慢慢的想融入到人类的感情中；伐木工虽然空有着人类的体格和智慧，

但是真正的皮囊之下却是污秽肮脏的思想和躯体。

一个只有人类皮囊的黑炭人，却被一个拥有健康体格和人类思想的人类猥亵甚至意图强奸，

那些披着人皮的人类啊，你们的头颅下面到底是什么啊。

本以为，斯嘉丽会被摩托男干掉，看来被人类干掉，整个影片升华的空间，提高了一大截啊。

至于摩托男来自哪里，什么身份，又要去哪里，不管不问！

恐怕是冲着宣传的全裸的斯嘉丽来的吧，也不动脑子想想，经过广电那帮太监审核过的片子，
