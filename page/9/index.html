我看小王子

发表于 2016-03-04 分类于 谈天说地 Disqus：

是的，每个人心里都有一个小王子！

我和老婆刚认识的时候她就给我推荐过小王子，那个时候还没有电影呢，可惜一直没有放在心上，也一直没有特意的看书，于是趁着电影上映的时候陪着老婆和娃子去看了一遍。

看完之后，确实被感动到了，应该说：每个人心里都有一个小王子。

我们还能否记起小时候的英雄梦，是否还记得幼时的横冲直撞，是否还记得幼时的天真烂漫，是否还记得小时候的那些玩具朋友。

有时候，我们是长大了，但是长大的过程中最可怕的不是年龄的增长，而是遗忘。很多时候，我们不是因为长大了而变得功名利禄，而是我们把有些东西遗忘了。忘在一个角落里再也想不起来了。

如同小王子的玫瑰，小王子曾经是那么的深刻的爱着，可是当小女孩去找王子的时候，他还是想不起来玫瑰是什么了！

那我们是否为了生存，为了面子，为了成长，为了金钱，为了工作，而忘记自己也曾经有过一朵玫瑰呢？

其实，一直到现在，我都还不知道作者的名字，看电影的过程中我想，小王子的作者该有多孤单啊！

想在成人的社会规则中保持自己的童心，想在成长之后保持自己的玫瑰，可是他不能被社会所接受。被听到掌声就脱帽致谢的自大狂鄙视，被自以为是高高在上的电梯员鄙视，被唯利是图的老板鄙视。

他不能被世人理解，也许当初曾努力的保持自己的童心，可是发现在纷扰的尘世中，根本无法继续生存下去。所以他接受了所谓的帮助和教育。

那些人把他教育成了现实中需要的人，一个不停打扫烟囱的好员工-王子！

他忘记了自己是谁，忘记了小狐狸是谁，忘记了玫瑰是谁，甚至忘记了星星，忘记了绵羊。

那些可怕的大人！

有时候，我会想，也许作者本人就是那个飞行员老头，童年里有着飞行员的梦想，还喜欢每天在夜晚里看星星，或许他在乡下或农场里真的有一个狐狸好朋友，也许他第一次看到玫瑰花的时候，真的就深深的爱上了她，可是玫瑰刺伤了它，也许小的时候幼儿园的老师让他画绵羊的时候他就真的画了一个盒子，也许他曾看着天空中飞过的鸽子，幻想过让鸽子带着他飞过天空，飞到很远的星球，他甚至都给好多好多的其它星球起了名字和编号，其中一个就是长满猴面包树的星球，还有他的玫瑰也在那个星球上，还要不停的把猴面包树锄掉，不然就会变得很糟糕。

他的脑子里长满了各种各样奇怪的想法。

可是他必须要长大，他也要上学，也许他上的就是沃茨学院，学校把他按照标准规格制造成了合格的的大人，他有工作，有朋友，有生活。只是他没有了童年。

他也许曾经试图保持自己的纯真，可是一丝不苟的警察，自大狂，电梯员，老板和生活一次又一次的把他打回了现实。

他必须保持不停的工作，对于大人来说工作就是一切，只有工作才能获得大人的认同。每当一个小王子长成大人时，他的星星就被封闭在那个罐子里了，不但丢失了童年的一切，也丢失了找回童年的信心。

在大人的世界里，会不停的有人提醒你：你是一个大人，做你该做的！

是的我们必须要做我们该做的，不管为了什么，我们都应该去做。但是在成人的世界里，我们也许可以偷偷的保留一点童心。

它可以时刻提醒我们，在面对孩子那些莫名其妙的想法，和不合规矩的举动时，应该蹲下来和孩子说话，而不是高高在上的怒吼，让孩子仰望你生气愤怒的脸庞。

孩子仰望的应该是浩瀚的星空。

小王子也许每两年就该看一次，也许每次看的心情和体会都不同。

娃子现在五岁了，再过两年就该上小学了，那时她就有自己的玩伴和更多的想法了，目前来说她还属于天真烂漫的年龄，还没有作业考试的烦恼，还没有青春躁动的无处安放，更没有安家立业的压力，这是她最无忧无虑的时间，也许现在是她最接近小王子的状态。我希望能在她最好的季节里陪她经历生活中的每个细节，知道她每次的喜怒哀乐，陪着她渐渐长大，保护好她的小王子。

我的小王子已经丢了，再也找不回来了。

因为我是大人了！

霾

发表于 2016-03-01 分类于 谈天说地 Disqus：

并不是每个人都知道霾的滋味，霾是一种病毒，总是在你身上那微小的伤口处生根发芽，并慢慢的成长。

也许霾是一种流行病，而且无药可救，有的人感觉一塌糊涂，有的人感觉舒服刺激，有的人已经麻木。当霾压过人们头上的天空时，人们只会用眼轻轻的扫一下，又接着忙自己的生计去了。

霾四处蔓延着，冲破了人们的心理防线，静悄悄的在人群中落地生根，像恶魔一样控制着人们的心灵，驱使着人们的驱壳。人们只有在霾中出生，在霾中生存，然后在霾中死去。不再为庸俗杂事而烦心，不再为自己奔走呼啸。

霾延伸着各种不堪的情绪，睡了，醉了，哭了，痛了。每个人身上的霾的滋味都不一样，而感觉却是相似，一种失落、惆怅、迷茫、彷徨、怀疑、忐忑，充斥着世界的每个角落。

当霾笼罩着大地，所有习惯于阴暗中生存的东西都出现了，人们被霾笼罩着，心灵被黑暗控制着。

当雾来临，人们显出阳光的一面，勤劳、勇敢、诚实、善良，以及世界上最美好的东西；而当霾出现，一切丑陋的本性又出现在人们的脸上，贪婪、污浊、卑鄙、阴险，每个人都生活在双重性格下。

所以，你不必为你的骄人成绩而沾沾自喜，也不必为你的大错已铸而嗟叹不已。 人既要活在雾中，也要活在霾中。

大雾轻起的时候，人们不得不把自己的丑陋隐蔽，用阳光的词汇装饰在脸上。如同嗜血如命的吸血鬼，把自己肮脏的嘴脸隐藏在看不见的霾中，面对着甜美的血液却无法伸手去触摸，那种急切渴望，口中流着馋涎，不停四处游荡。压抑心中的欲望，时刻想要冲出去，即使灰飞烟灭也在所不惜。但最终却只能隐藏在霾中，眼巴巴的望着鲜血，期望着太阳永远的死去，世上从此都是霾。

大地初开，便有光，有光就必有影，光和影是同生同灭的，无暗无光，无光无暗，任何事物都时时刻刻在受着光和影的影响，光的出现，也许就是为人们遮掩丑陋的。

就如同人同时生活在雾和霾中。有的时候为了自己的正面形象，必定要有雾，但是为了自己的私欲，又必然需要霾，因为只有在霾中才敢揭去雾的面具，成为一个真实的自己。

人活一世，就必然有面子和里子，面子有多光鲜，里子就有多肮脏。

然后摩托男背出来一个女尸，斯嘉丽全裸出演，脱女尸的衣服给自己穿上，脱完之后，女尸的眼角竟然流出了一滴眼泪。也许是感叹人生，也许仅仅是感叹自己死后还要被折腾一番。

她想成为的人类把她杀了。

本以为，斯嘉丽会被摩托男干掉，但是最终被人类干掉，觉得整个影片升华到了另外一个高度！摩托男来自哪里，什么身份，又要去哪里，根本不重要！

至于网上说的，看不懂，没意思，恐怕只是冲着网传的全裸的斯嘉丽来的吧，也许你也可能看不懂《第九区》。其实想看裸体的话何不去看岛国的动作片和棒子国的情色片，比这撩人多了。

我认为这毕竟只是一部文艺片。

写教程是真难！

发表于 2015-07-26 分类于 Swift Disqus：

第二章终于写完了，原本打算 3 天写完，结果实际上花费了 7 天时间。

一般我的步骤是按照英文教程先编一遍应用，出来一个可以跑的应用。然后再

按照教程，再来一遍代码，在代码的过程中截图、写作，发现重心全放到文章的梳理和

协调上去了，根本对编程内容和技巧没有一点提高。

所以，改变了方法，第一遍还是按照英文教程编一个应用，

第二遍是细读，根据读得内容写文章，这个过程不牵涉到写代码和截图。

第三遍是写代码和截图。

这样的话，三遍下来，一章教程 300 页，第一遍基本是 6-7天，第二遍4-5天，第三遍基本是10天。

也就是说，300 页的教程要20天才能完成，现在真的觉得那些用心写教程，写书的人不容易。

在这方面做的比较牛逼的一个是李笑来，一个是阮一峰，他们的文章和出版的书，在他们的博客里都

有，你当然可以不花钱就能够读到。现在我觉得，如果觉得别人的劳动对你有所帮助，有价值，

还是掏钱买正版书籍看吧，一来养眼益目，二来是对别人劳动的尊重。

在第二章中，后半段纯粹就是应付差事了，把代码拿来输入一遍，也没有过脑子，也没有带 Xcode 7

中测试，需要 swift 2 版本的代码到 github 上下吧。
