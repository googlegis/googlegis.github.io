如何在mac下修改path中的环境变量

发表于 2014-08-27 分类于 Mac

How To Edit Your PATH Environment Variables On Mac OS X

If you are new to Mac OS X, you may need to know how to edit your PATH. The good news is that this is an easy task on Mac OS X.

The recommended way is by editing your .bash_profile file. This file is read and the commands in it executed by Bash every time you log in to the system. The best part is that this file is specific to your user so you won’t affect other users on the same system by changing it.

Step 1: Open up a Terminal window (this is in your Applications/Utilites folder by default)

Step 2: Enter the follow commands:

touch ~/.bash_profile; open ~/.bash_profile

This will open the .bash_profile file in Text Edit (the default text editor included on your system). The file allows you to customize the environment your user runs in.

Step 3: Add the following line to the end of the file adding whatever additional directory you want in your path:

export PATH=”$HOME/.rbenv/bin:$PATH”

That example would add ~/.rbenv to the PATH. The $PATH part is important as it appends the existing PATH to preserve it in the new value.

Step 4: Save the .bash_profile file and Quit (Command + Q) Text Edit.

Step 5: Force the .bash_profile to execute. This loads the values immediately without having to reboot. In your Terminal window, run the following command.

source ~/.bash_profile

That’s it! Now you know how to edit the PATH on your Mac OS X computer system. You can confirm the new path by opening a new Terminal windows and running:

echo $PATH

You should now see the values you want in your PATH.

[Updated 2/10/2012] The instructions now use the .bash_profile method of editing your PATH. This is preferred as it keeps the changes specific to your user. I also updated the instructions to use Text Edit instead of vim so it is easier for a beginner.

我得了脂肪肝

发表于 2014-08-14 分类于 文章

病从气上出，不出所料，得病了。这个是我想都不敢想的病，竟然会得脂肪肝。

总结一下可能的几个原因：

  1. 心情：已经很长一段时间没用开心过了。心情是极度的压抑和抑郁。

  2. 饮食：自从丈母娘来到之后，老婆想着每天都要吃点好的，所以基本上每天晚上
    都要吃肉，而且一吃就是吃三年，如果白天上班的时候再吃肉的话，基本上每天就是两顿肉，天天如此，饮食结构坏掉了。而且丈母娘有个坏习惯，她以为你吃的多，吃撑了是好事，所以每次必做多，每次吃饭必劝饭，很多时候，为了照顾她的颜面，不自觉的会多吃了一点；她认为如果不做肉吃，炒菜多放油也是能够提供营养的，所以无论炒什么菜，都是油当先。
    也怪自己立场不定，为了照顾他人颜面，必定会对自己造成伤害。

  3. 运动：锻炼都是断断续续，没有坚持下来，也从来没有考虑过自己会得病。跑步跑了一段时间，没有坚持下来，仰卧起坐一段时间，没有坚持下来。好多时候都是自己放弃了。都是自己放弃了。

书到用时方恨少，病到身上才知多。

决定从几个方面做改善：

  1. 心情：把工作和锻炼放在第一位，不把注意力放在生活琐事上，这样就能转移注意力。必要的时候才会专门讨论这些事情。

2.饮食：坚持健康的饮食习惯，坚持自己的原则。不因为别人的言语而改变。

3.运动：暴走，从家里去公司约5公里，每天来回一次10公里。周末到世纪公园来回15公里。

一个月后看情况。

Fix Chrome’s Voice Search has been turned off’

From Chrome menu select Settings, select Show Advanced Settings

Privacy>Content Settings scroll down to ‘Media’ make sure ‘ask me when a site requires access to my camera and microphone (recommended)’ option is selected (check the screenshot below) instead of this one ‘Do not allow sites to access my camera and microphone.’

Click ‘Manage Exceptions’ and remove any exceptions you might’ve added for Google site in past which showing Deny words for Audio and click ‘Done’ button to save the changes.

Advanced Settings Media thumb Fix Chrome is Showing Voice search has been turned off. Details Error

Now try to start voice search in Chrome and it works as expected without showing any errors. FYI, ‘Ok Google’ has came to desktop Chrome, you need to install Google Voice Search Hotword beta extension for that. After installing that extension you can talk to Google without clicking on microphone icon or you just need to say ‘ Ok Google’ then on Google homepage.

关于 mac的几点想法

发表于 2014-06-24 分类于 文章

今年的春天，买了RMBP 2013later，32岁的年龄，还是觉得要冲一下，在有一定基础的情况下，做一次自己喜欢的事情。

###首先第一条，mac的笔记本到底贵不贵？

贵不贵看情况，如果你都是在windows下的开发，而且对mac的开发一点兴趣没有，买个mac就是为了装13的，那mac真贵，贵到你连看都不想看。随便一个本子就MBA也的7000+，就不说联想了，dell和hp的机器也就3000-4000配置就很好了嘛。

然后再看配置，首先一个ssd的硬盘，512G去jd上搜一下，看看多少钱，再看看ms的surface，512G的ssd多少钱?然后再看mac是不是贵，是不是值。

###第二条，mac上面什么都不能做啊？

那么多exe的软件在mac下鸟用没有，你让我怎么办？我想装个网银都不能用，还有360的杀毒也不能用，那要是中毒了怎么办？亲，选对的，别选贵的。

###第三条，mac下我能做什么呢？

对于普通用户来说，真的什么都做不了，就算打开个word还要转换成pages下面才能编辑，而且保存后的文档别人打不开，需要转成word别人才能用，这不是花钱找罪受吗？

如果你确实生活在windows的圈里出不来，还是不要出来了。

###第四条，mac的用途:

####1. 连接iPhone，牛叉。
你会觉得以前在windows下载安装iTunes，就像是带套去ML，虽然也是嗯嗯呀呀的费半天劲，但是总是感觉少了一点什么？到底是什么呢？少了一点速度和激情。

####2. 超高的分辨率，
这个只是针对我以前的那个旧pc来说的，换了之后，感觉整个世界都清晰了，连上楼都有劲了。

####3.完整的办公及娱乐套件
iWork 和 iLife 的应用，够你用的了。而且现在买的mac里这些应用都是免费的。对了，不说到想不起，你的windows下的msoffice 是正版的吗？

####4. 免费的系统
这个我还真不知道从哪个osx的版本开始的，我开始用的时候就10.9 了。反正一直都是免费，这个对windows来说，简直就是天大的好事，连升级都要收费，吃多了吧。不过windows最近也开始学乖了，推出了一系列免费及优惠政策。

####5. 商店 iTunes Store 及 APP Store
以商店模式进行应用的索引及排名，提供开发机会。
这样的模式对开发者来说再好不过，只要你有创意，有技术，还是可以站着把钱挣过来的。(国内的那些歪门邪道暂且不表)

####6. 牛X的iPhoto
为什么单独把它提出来说，在买这个mac之前，我在windows下还真没有看到哪个应用或者程序能够做的如此友好而且精巧，不得不表彰一下。

####再比如其他的基于linux、安全、性能优化、wifi共享…… 可惜我不是技术流，对这些都是知，而不解。只能提提而已了。

对任何事情都有个先入为主的概念，比如对初恋，比如对第一部手机，比如对windows，如果你刚开始接触电脑时，就开始使用osx系统，而不是windows，恐怕现在是另外一种想法了。

所以换不换，用不用，全在你一念之间。
