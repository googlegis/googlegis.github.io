我得了脂肪肝

发表于 2014-08-14 | 分类于 爱就趁现在 |

病从气上出，不出所料，得病了。这个是我想都不敢想的病，竟然会得脂肪肝。

总结一下可能的几个原因：

  1. 心情：已经很长一段时间没用开心过了。心情是极度的压抑和抑郁。

  2. 饮食：自从丈母娘来到之后，老婆想着每天都要吃点好的，所以基本上每天晚上
    都要吃肉，而且一吃就是吃三年，如果白天上班的时候再吃肉的话，基本上每天就是两顿肉，天天如此，饮食结构坏掉了。而且丈母娘有个坏习惯，她以为你吃的多，吃撑了是好事，所以每次必做多，每次吃饭必劝饭，很多时候，为了照顾她的颜面，不自觉的会多吃了一点；她认为如果不做肉吃，炒菜多放油也是能够提供营养的，所以无论炒什么菜，都是油当先。
    也怪自己立场不定，为了照顾他人颜面，必定会对自己造成伤害。

  3. 运动：锻炼都是断断续续，没有坚持下来，也从来没有考虑过自己会得病。跑步跑了一段时间，没有坚持下来，仰卧起坐一段时间，没有坚持下来。好多时候都是自己放弃了。都是自己放弃了。

书到用时方恨少，病到身上才知多。

决定从几个方面做改善：

  1. 心情：把工作和锻炼放在第一位，不把注意力放在生活琐事上，这样就能转移注意力。必要的时候才会专门讨论这些事情。

2.饮食：坚持健康的饮食习惯，坚持自己的原则。不因为别人的言语而改变。

3.运动：暴走，从家里去公司约5公里，每天来回一次10公里。周末到世纪公园来回15公里。

一个月后看情况。

谁能想到，世界上还有人敢菜巴西7个球。

听闻有足球世界杯这回事，知道那个时候有个贝克汉姆，知道德国的三驾马车已经

没落。然后就是整个高中时期。

看球看了个天昏地暗，那个时候的德国队应该是属于巴拉克和卡恩的年代。没落了。

但是也就是从卡恩和巴拉克身上开始喜欢德国队，坚韧刚强。

2006年正好赶上事业的上升期，把世界杯这茬给忘了，翻资料来看，那个时候，

拉姆、波多尔斯基、博罗夫斯基、施奈德等已经在和巴拉克/卡恩那个阶段的人开始

交替了。

2010年世界杯，就会发现，德国队的阵容完全变了，这里面只有克洛泽是老人了，

但是他永远都是一个年轻的面容，诺伊尔/拉姆/施魏因斯泰格/厄齐尔/戈麦斯/穆勒，

这些现在牛逼的人2010年就出现在世界杯上，那时的我也在想，这不是娃娃军团吗？

这来打什么球啊？明知他们不会走很远，但是还是默默的注视着他们，等4年之后吧。

4年之后这些人就牛逼了。

2014年世界杯，上届世界杯上的那些人又回来了。

踢巴西 7:1, 看着巴西球迷的哭泣，真的感到可怜，可是足球就是战场，没有人会怜悯

你，就像当年没有人会怜悯德国队一样。

联想到前面已经走的那几个球队，西班牙/英格兰/意大利/法国，有的还是老弱病残占

据主力不肯下来，有的则是缺乏血液循环，后续跟不上，再则是新手还没有大赛经验

需要时间和经历来磨练。

这么看来，后面无论是阿根廷或者荷兰，还真的不是德国的个了。

好吧。德国队，这次你牛逼起来吧。也不枉我支持你12年了。

已经走了的那几个国家队，回去更换血液，积累经验，下次再来吧。

巴西，这个国家不想说，从小罗不踢的那个时候就开始走下坡路，足球王国也毕竟是

皇帝的新装，缺一两个人整个球队就溃不成军，德国队则是那个说实话的小孩。

chrome voice search has been turned off

Fix Chrome’s Voice Search has been turned off’

From Chrome menu select Settings, select Show Advanced Settings

Privacy>Content Settings scroll down to ‘Media’ make sure ‘ask me when a site requires access to my camera and microphone (recommended)’ option is selected (check the screenshot below) instead of this one ‘Do not allow sites to access my camera and microphone.’

Click ‘Manage Exceptions’ and remove any exceptions you might’ve added for Google site in past which showing Deny words for Audio and click ‘Done’ button to save the changes.

Advanced Settings Media thumb Fix Chrome is Showing Voice search has been turned off. Details Error

Now try to start voice search in Chrome and it works as expected without showing any errors. FYI, ‘Ok Google’ has came to desktop Chrome, you need to install Google Voice Search Hotword beta extension for that. After installing that extension you can talk to Google without clicking on microphone icon or you just need to say ‘ Ok Google’ then on Google homepage.

关于 mac的几点想法

发表于 2014-06-24 | 分类于 文章 |

今年的春天，买了RMBP 2013later，32岁的年龄，还是觉得要冲一下，在有一定基础的情况下，做一次自己喜欢的事情。

###首先第一条，mac的笔记本到底贵不贵？

贵不贵看情况，如果你都是在windows下的开发，而且对mac的开发一点兴趣没有，买个mac就是为了装13的，那mac真贵，贵到你连看都不想看。随便一个本子就MBA也的7000+，就不说联想了，dell和hp的机器也就3000-4000配置就很好了嘛。

然后再看配置，首先一个ssd的硬盘，512G去jd上搜一下，看看多少钱，再看看ms的surface，512G的ssd多少钱?然后再看mac是不是贵，是不是值。

###第二条，mac上面什么都不能做啊？

那么多exe的软件在mac下鸟用没有，你让我怎么办？我想装个网银都不能用，还有360的杀毒也不能用，那要是中毒了怎么办？亲，选对的，别选贵的。

###第三条，mac下我能做什么呢？

对于普通用户来说，真的什么都做不了，就算打开个word还要转换成pages下面才能编辑，而且保存后的文档别人打不开，需要转成word别人才能用，这不是花钱找罪受吗？

如果你确实生活在windows的圈里出不来，还是不要出来了。

###第四条，mac的用途:

####1. 连接iPhone，牛叉。
你会觉得以前在windows下载安装iTunes，就像是带套去ML，虽然也是嗯嗯呀呀的费半天劲，但是总是感觉少了一点什么？到底是什么呢？少了一点速度和激情。

####2. 超高的分辨率，
这个只是针对我以前的那个旧pc来说的，换了之后，感觉整个世界都清晰了，连上楼都有劲了。

####3.完整的办公及娱乐套件
iWork 和 iLife 的应用，够你用的了。而且现在买的mac里这些应用都是免费的。对了，不说到想不起，你的windows下的msoffice 是正版的吗？

####4. 免费的系统
这个我还真不知道从哪个osx的版本开始的，我开始用的时候就10.9 了。反正一直都是免费，这个对windows来说，简直就是天大的好事，连升级都要收费，吃多了吧。不过windows最近也开始学乖了，推出了一系列免费及优惠政策。

####5. 商店 iTunes Store 及 APP Store
以商店模式进行应用的索引及排名，提供开发机会。
这样的模式对开发者来说再好不过，只要你有创意，有技术，还是可以站着把钱挣过来的。(国内的那些歪门邪道暂且不表)

####6. 牛X的iPhoto
为什么单独把它提出来说，在买这个mac之前，我在windows下还真没有看到哪个应用或者程序能够做的如此友好而且精巧，不得不表彰一下。

####再比如其他的基于linux、安全、性能优化、wifi共享…… 可惜我不是技术流，对这些都是知，而不解。只能提提而已了。

对任何事情都有个先入为主的概念，比如对初恋，比如对第一部手机，比如对windows，如果你刚开始接触电脑时，就开始使用osx系统，而不是windows，恐怕现在是另外一种想法了。

所以换不换，用不用，全在你一念之间。

带孩子的几点想法

发表于 2014-06-05 | 分类于 文章 |

1. 不要总是盯着缺点

丈母娘从2011年9月份来上海，照顾老婆然后一直照顾小孩，老年人的习惯加上老家的思想，总是感觉处处

不如意，时不时的就莫名的来火上气，总感觉这做的不像样，那也做的不像样，什么都做不好。

有时候，自己脾气上来就给别人脸色看，老婆不高兴，丈母娘不高兴，连着小娃子都要受委屈。

这次独自带娃子去武汉参加同学聚会，真的是感觉到了带小孩的辛苦。

吃饭喝水，拉屎撒尿，闹人睡觉，嘘寒问暖，样样要操心，样样要做到，我自己做来还破费周折，况且还没

有其他人在跟前看着，更没有人会对我检查。即便这样，我还是不能随心应手，更何况一个老年人，照顾自

尚且有困难，还加上一个不怎么听话的娃子，还担心女婿晚上回来看到没有做到的地方会生气。

:-(

我犯了大错了。

前段时间，总是想找个人问问，该如何缓解和丈母娘之间的这种对立的情绪。

这下自然的消除了一部分，虽然有时还是会看到做的不对的生气，但是自我感觉没有以前的那种恨铁不成钢

的气势了，毕竟是老年人了，而且，自己又不是亲儿子，娃子又不是亲孙女，刷锅做饭，洗衣倒水，还不都

为了她闺女，我媳妇吗。

站在媳妇的角度，站在丈母娘的角度，再看看，所有的事情都不值得生气了。

娃子慢慢的长吧。

希望不要丢失了自己的灵性。

2.小孩是要哄的

小娃子哭，小娃子闹，小娃娃天天睡大觉。我的脾气不好，估计和最近的工作压力有关，也和这行的工作

性质有关，喜欢一个人安静的听从自己的安排。电脑听自己的，代码听自己，周围的人都听自己，不允许

有瑕疵，有bug，(话说我是什么星座？)，但是小娃子就不一样了，小娃子有个性，会思考，会讲理，自己

有腿有脚，会自己跑，哪能事事听你的啊。

有时候自己的脾气上来就用强力制止小娃子，小娃子就会用哭来抵抗，自己还总是训她不听话，两败俱伤。

其实你换个方法也许更好，比如买个糖，买个玩具。

如同聚会的时候，如果她不听话，买个糖或者玩具给她，她就会老实下来。

娃子是个吃货，

不要和她较劲，用个糖哄一下，比你教育的效果好。

3.难题

一直都是教育娃子过马路，红灯不能过绿灯能过，娃子也很听话，平时总能遵守规则，时不时带她出去

玩的时候还测试她：这个是什么灯，能不能过？

娃子总能按照红灯停绿灯行的规则回答。

小区后面有个大马路，有拐弯灯，平时过马路时，车辆直行灯变红时，拐弯灯变绿，但是人行道上灯还

是红色的。但是大家根本等不及人行道灯变绿，就直接闯红灯过去了。

有次带她出去玩，回来晚了，当时也没有太注意，正好赶上拐弯灯变绿，就问娃子：现在能不能过？

娃子说：能过！

“那现在不是红灯吗？怎么能过呢？”

“别人都能过了！”

我脑子停顿了一下，别人都能过了，该怎么解释呢？

“别人都不是乖宝宝，上幼儿园回来老师不发小红花，娃子是乖宝宝，上幼儿园老师发小红花。”

出一头汗，总算混过去了。

估计以后还会多次碰到这种情况，别人为什么能插队？别人为什么能受贿？别人为什么能偷钱？

:-(

等她再长大一点慢慢解释吧。

4. 智商

出去玩回来第二天，妈妈有几个口香糖放在桌子上，不知道怎么让娃子看到了，拿到手里不肯松开，

要给她夺下来，还和你讲理：俏俏又不吃，俏俏拿着。

不过那个口香糖的味道确实香，娃子手里掐了一点，放在鼻子上闻闻，放在嘴边舔舔，你一看她，她

就说： 俏俏又不吃。

好，拿着就拿着吧。

妈妈下班了，带她出去接妈妈，外面想下雨，拿着伞，娃子手里还是拿着口香糖。

出了小区，过马路，娃子双手搂着我的脖子，等她回过脸的时候，看她一脸神秘的微笑，我看口香糖上

已经有口水的印子了，问她：你是不是吃口香糖了。

娃子只是笑，过了一会，娃子说：爸爸你打伞吧，下雨了。

其实在树下走，天空还没有开始下雨，

“没有下雨啊”

“你看，都滴到俏俏的脸上了”

“在哪里？” 我问，

“你看你看，”

她这么一说，我还真觉得有点下了，我当时是一个胳膊抱着她，一个手拿着伞，还得用另外一个手帮着

撑开伞，就在我撑开伞的那一个瞬间，我看见娃子的手，迅速的把口香糖放到嘴里，然后快速的闭上嘴。

我连忙从她嘴里把口香糖扣出来，她自然是不愿意，哭得稀里哗啦，

我想既然哭了，索性把手里的都扔了吧。她双手攥的紧紧的，哭得昏天暗地。她去打疫苗时都没有哭。

小娃子，这个东西你不能吃。

等她稍微缓了点劲，告诉她，“有些东西可以吃，有些东西不能吃，棒棒糖可以吃，口香糖不能吃。”

最后买了个棒棒糖哄她，才停止啜泣。

晚上的时候，在想，小娃子告诉我下雨了，让我去打伞，是不是早就准备好了要吃口香糖呢，

她是不是知道我一直盯着她的话，她就不能吃到嘴里，必须要趁我不注意快速的吃掉呢。

那她是怎么想起让我去打伞的这个机会的呢？

这么说来整个事情都是小娃子的预谋了。

两岁半的小娃子就能这么思考(也许是本能)，比你爹强。
