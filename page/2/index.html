七年之痒

发表于 2017-03-06 | 分类于 文章 |

只见我方这名后腰队员，一个人带着两个球，成功突入对方的防线，进入了小禁区。对方上来两名后卫进行拦截，这名队员左闪右躲，凭借着精壮的体力和娴熟的技巧，一脚角度极刁的香蕉球，成功的避开了守门员的视线，终于把球给射进了。

全场的球迷都欢呼起来，

出轨嫖娼队又得一分，陈思成单膝跪地，拉弓射箭，直上云霄。

文章、陈赫、王全安…这些队友跑过去搂住他紧紧的，紧紧的拥抱在一起，空中传来他们欢呼的声音！

是的，网传陈思成出轨了，又一个男人没有勒紧自己的裤腰带。

其实不只是娱乐圈的男人没有勒紧自己的裤腰带，好像无论哪个朝代总有些男人裤腰带会松的掉下来，从古代的文人骚客，到今天的明星演员，每个时代总有一些人过不了这个关！

有的时候我就想，你都这么有钱了，这么出名了，老婆都那么漂亮了？为什么还要搞嫖娼劈腿的事呢？

后来我想明白了，妻不如妾，妾不如偷。正是他们有名有权了，给办事提供了方便，正是因为有钱了，才不会忘乎所以，不为衣食而忧。

★ 明·凌濛初《二刻拍案惊奇》卷二十八
这个程朝奉拥着巨万家私，所谓饱暖生淫欲，心里只喜欢的是女色。

★明·凌濛初《二刻拍案惊奇》卷二十一
自古道：“饱暖思淫欲”，王禄手头饶裕，又见财物易得，便思量淫荡起来。

饱暖思淫欲，说的就是这些货，吃饱了，穿暖了，嘴里叼个牙签舒舒服服躺在那里的时候，脑子开始滴流滴流的乱转想歪主意了。

你要是平头百姓，出去搬一天的砖，挣的钱都不够一家人吃饭的，回到家躺在床上还要听老婆唠叨，恐怕也没有时间和精力考虑这些下半身的问题了。

只有当脑子分泌的荷尔蒙过量的时候，才会拿下半身来思考。 可是在你下半身思考之前，你脖子上的那个是葫芦吗？

曾经想过这个问题：如果人类仅仅是靠情感来控制自己，不需要家庭道德和法律约束，那人与人将会组成怎样的社会关系？

每个人都会有相恋相爱，然后发现不合适，直接走开与另外一个人结合，然后发现更好的。每个人都在晋级，每个人都在挑选更好的。

自己选择别人，然后被别人选择；自己抛弃别人，然后被别人抛弃。年轻的人一茬又一茬，年老的人在慢慢退出这个舞台，甚至于，各个年龄段的都在进行选择和抛弃？然后力量强大的男人可以多占用几个好看的女人；力量强大的女人可以多养几个健硕的男人。然后人们就生活在生存、斗争、和交配权的争夺上，最后，人们都不需要什么精神享受和道德约束了，也不需要多余的衣食住行了，我们就尽情的非洲草原上撒欢了。

这时耳边响起了赵忠祥雌性的声音：又到了一年一度交配的季节，人群在草原上尽情的奔跑着，在为这次的狂欢做着准备。

我艹，不忍直视！

猴子费了千万年的时间，进化成了可以使用工具的人类，现代人类只用几十年甚至几年就退化为只用下半身思考的畜生。

七年之痒，是每个家庭都要过的槛，掩耳盗铃总不是解决问题的方法。如果一个人想用下半身思考人生，躲都躲不过；如果不想，你把它丢到一堆裸体里面也能全身而退。

所谓的七年之痒，关键是看你是上半身痒还是下半身痒。

霾

发表于 2017-03-01 | 分类于 文章 |

并不是每个人都知道霾的滋味，霾是一种病毒，总是在你身上那微小的伤口处生根发芽，并慢慢的成长。

也许霾是一种流行病，而且无药可救，有的人感觉一塌糊涂，有的人感觉舒服刺激，有的人已经麻木。当霾压过人们头上的天空时，人们只会用眼轻轻的扫一下，又接着忙自己的生计去了。

霾四处蔓延着，冲破了人们的心理防线，静悄悄的在人群中落地生根，像恶魔一样控制着人们的心灵，驱使着人们的驱壳。人们只有在霾中出生，在霾中生存，然后在霾中死去。不再为庸俗杂事而烦心，不再为自己奔走呼啸。

霾延伸着各种不堪的情绪，睡了，醉了，哭了，痛了。每个人身上的霾的滋味都不一样，而感觉却是相似，一种失落、惆怅、迷茫、彷徨、怀疑、忐忑，充斥着世界的每个角落。

当霾笼罩着大地，所有习惯于阴暗中生存的东西都出现了，人们被霾笼罩着，心灵被黑暗控制着。

当雾来临，人们显出阳光的一面，勤劳、勇敢、诚实、善良，以及世界上最美好的东西；而当霾出现，一切丑陋的本性又出现在人们的脸上，贪婪、污浊、卑鄙、阴险，每个人都生活在双重性格下。

所以，你不必为你的骄人成绩而沾沾自喜，也不必为你的大错已铸而嗟叹不已。 人既要活在雾中，也要活在霾中。

大雾轻起的时候，人们不得不把自己的丑陋隐蔽，用阳光的词汇装饰在脸上。如同嗜血如命的吸血鬼，把自己肮脏的嘴脸隐藏在看不见的霾中，面对着甜美的血液却无法伸手去触摸，那种急切渴望，口中流着馋涎，不停四处游荡。压抑心中的欲望，时刻想要冲出去，即使灰飞烟灭也在所不惜。但最终却只能隐藏在霾中，眼巴巴的望着鲜血，期望着太阳永远的死去，世上从此都是霾。

大地初开，便有光，有光就必有影，光和影是同生同灭的，无暗无光，无光无暗，任何事物都时时刻刻在受着光和影的影响，光的出现，也许就是为人们遮掩丑陋的。

就如同人同时生活在雾和霾中。有的时候为了自己的正面形象，必定要有雾，但是为了自己的私欲，又必然需要霾，因为只有在霾中才敢揭去雾的面具，成为一个真实的自己。

人活一世，就必然有面子和里子，面子有多光鲜，里子就有多肮脏。

我看爱乐之城

发表于 2017-02-28 | 分类于 看电影 |

奥斯卡就要举办一年一度的盛典了，作为奥斯卡的冲奖热门《爱乐之城》，在国外受到了一片赞誉，至于国内的排片和票房，没有特意去搜索过，所以不得而知。但是我猜测票房不会太好，能及格就不错了，因为这只是小众的文艺片！

趁着周末休息的时候，去电影院看了一遍，感觉是近几年来不错的歌舞片，上个比较欣赏的一个歌舞片，好像是陈可辛的《如果 爱》。

歌舞片首先要保证歌曲舞蹈的无缝切入，不能前面都是演员演绎感情，背景音乐调动情绪，等把观众的情绪调动起来了，突然来了一段歌曲或者舞蹈，让观众出戏。

其实，就故事情节来说，《爱乐之城》的故事属于老套情节，心中充满某种文艺情节的底层小青年，一心要奋斗出自己的天空，在这个过程中，经历了无数失败和嘲讽，经历了巅峰和低谷，最终认识到了最爱的已经失去了。

这种励志的片子在好莱坞的影片中数不胜数，前有《新美国梦》，后有《魔鬼代言人》，无一不展示在底层奋斗的艰辛与困苦。

但是这部爱乐之城，就像是一杯值得细细品味的咖啡，静下心来，一个人去电影院，四周黑暗的氛围里，慢慢看，细细品。也许开始你不适应歌舞片的节奏，但是当一个场面宏大的马路牙子群歌群舞之后，你也许就能慢慢融入到影片里了。

江湖人称”石头姐”的艾玛-斯通，以前听说过这个人，但是没有从来没有看过她的影片，但从此片看来，称得上是演技派的老司机，举手投足中透露着自信，演戏对她来说应该是信手拈来的事。没有过多的化妆反而让观众看着更亲切，据说，影片中的大部分情节和石头姐本人的经历很类似。

高司令，在这部片子之前，也没有看过他的片子，只有在娱乐新闻中听说过他，我本来以为男人跳舞，肯定身体僵硬，舞步不协调，动作不漂亮，但是高司令让我改变了男人跳舞的偏见。

还有高司令弹钢琴的那段，据说为了演影片，自己苦练了三个月。上个让我因为弹钢琴而让我刮目相看的演员是《海上钢琴师》的蒂姆-罗斯，高司令那种弹钢琴的认真的态度，让我感受到了第一次看《海上钢琴师》时，随着钢琴的节奏而慢慢屏住呼吸的感觉。

影片中的双人舞是影片的亮点，无论是山顶的初识，还是热恋时的星空，都让人感觉是那么的美。

石头姐鹅黄暖色系的全色连衣裙，让人看着赏心悦目，估计应该是今年夏季的流行色。

不管如何的困难，俩人终究在一起，也同样是不管俩人如何努力，最终还是分开了。

虽然俩人最后都得到自己想要的东西，但是俩人之间再也没有交集了。只是导演的闪回给观众带来了无限唏嘘！

再相遇时，俩人只剩下了微微一笑。

这一笑，有人解读为无奈；这一笑，有人解读为满足；这一笑，有人解读为爱情。

我想这一笑，也许是为对方终于得到自己想要的艺术而衷心的祝福吧！

这一天，我去一个小电影院看时，影院里只有六个人，三个年轻的小伙子，加在一起年龄不超过一百岁，

另外三个人，是老年人，我猜年长的那位老奶奶应该超过七十多岁了，三个人加在一起应该有两百岁了。

很有幸，在这个小城市里还能遇到三位对《爱乐之城》感兴趣的长者。

学会向孩子道歉

发表于 2017-02-22 | 分类于 育儿经 |

今天早晨送娃子去幼儿园，刚到那里正准备放书包，班里的王老师，特意走到她身边说，

“我要和你说件事情，老师昨天家里有事了，忘记把画画的照片发给爸爸了，今天放学发好不好？”

小娃娃想了一下，

“好！”

这时，我才想起昨天下午放学接她的时候，发生的一件事情。

昨天放学接她，小娃子很兴奋，告诉我说，因为昨天画的画好，老师奖励给她糖吃，还有口袋里

有好多其他的好东西。

她问我，

“老师有没有把我画的画发给你看？”

我说没有啊。

她背着小书包，蹬蹬蹬的跑到王老师面前问，

”王老师，你有没有把我画的画发给爸爸看？”

王老师笑着说，

”还没有，今天放学回家后发好不好？”

小娃子同意了，然后和老师再见回家了，回到家又和家里人说了一遍画画的事情。

估计昨天老师太忙忘记了，所以就有了今天早晨老师给学生道歉的一幕。

其实我是很感动的，今天早晨去的时候，教室里已经来了很多同学，王老师还正在和园长说话，

看到小娃子进来，王老师还特意过来和她道歉，说明原因并达成一致意见。

我觉得这是一个伟大的老师。

其实对于每个家庭来说，自己的孩子就是全部，是100%；但如果对老师来说，一个家庭的孩子，

在30多人的班级里，只能算是3%。如果老师记住了和你的孩子说过的话，记住答应你的孩子的事情，

并因为没有做到而专门到你孩子面前商量道歉的话，能够像家长一样对待孩子，我是很感恩的。

不知道有没有家长从来没有和孩子道过歉，自己做错了也是做错了，但是对孩子没有任何表示，

不但没有‘我错了’这三个字，甚至连一个交流的机会都不给孩子。

有人认为，孩子是我生的，长这么大是我养的，钱是我出的，我给孩子道什么歉啊？

还有的人会认为，小屁孩一个，什么都不懂，道歉也没有意思。

其实，这些想法都是有偏见的，现在的小孩发育快，思想成熟早，在很小的时候，就有自己的想法了。

家长如何和孩子建立平等的关系，如果在孩子面前建立威信，全都是通过平时生活的点滴相处来

建立的。

你可以对孩子不道歉，可以对孩子大声恐吓，可以说话不算，可以辱骂甚至打骂家人，可以熬夜睡懒觉，

可以好吃懒做，可以抽烟酗酒，可以脏话连篇，这些都可以做，但是孩子也会学着做。

有的家长抱怨孩子打滚撒泼，有的家长抱怨孩子胡搅蛮缠，有的家长抱怨孩子磨蹭拖拉，但其实你

在与孩子的交往是否已经教给了她这些行为准则了呢？是否也做到了令行禁止，今日事今日毕呢？

是否蹲下来平等的与她对话，让她知道做好的结果，与做坏的结果的不同呢？

其实家长的一言一行影响着孩子的社会准则，家长的一举一动影响着孩子的生活习惯，这些都在孩子幼小的时候，

深深的印在脑海里，长时间的习性的积累，就会对孩子造成不同的影响。只有父母认真的管理好自己，

才能管理好孩子。

不以恶小而为之，不以善小而不为。

海边的小贝壳

发表于 2017-02-20 | 分类于 儿童故事 |

很久很久以前，在大海边有一个孤单的小贝壳，这里只有它一个人，虽然也有很多好朋友，可是

它从来没有到过其它地方，只能随着海水，飘过来，飘过去。它好想到其它地方去看看啊。

有一天，太阳还高高的挂在天上，海边来了一群小朋友，他们光着脚丫，在海边的沙滩上，从远处慢慢的

走过来，有的小朋友拿着小铲子，有的小朋友拿着小水桶，他们一边走着，一边说着笑着，

有时还会弯下腰来捡贝壳，放到桶里。

小贝壳就想啊，要是能捡到我该多好啊，这样我就能到其它地方去看看了，

走过一个小朋友，小贝壳就喊着，‘快捡我，快捡我’，

又走过一个小朋友，小贝壳又喊着，‘快捡我，快捡我’，

可是走过了一个又一个小朋友，没有谁愿意捡它，因为小朋友喜欢捡那些，大的，漂亮的，有颜色的贝壳，

可是小贝壳，不但小，还是普通的土灰色，根本没有小朋友愿意捡它。

小贝壳都有点伤心了，突然一只脚踩在了小贝壳的身上，它都陷到沙子里面了。

那只脚慢慢的抬起来，有只手把它捡起来，用力的扔到海里。

小贝壳就这样在大海里漂来漂去，漂来漂去，到了晚上还在漂，小贝壳都睡着了。

第二天，早晨太阳出来了，小贝壳睁开眼睛一看，自己真的到了另外一个地方了。

旁边有认识的朋友，也有不认识的朋友。

小贝壳，左看看，右看看，突然，它看到自己的爸爸妈妈？

小贝壳就大喊起来，

“爸爸， 妈妈，你们怎么在这里啊？”

“我们是被一个小朋友捡到之后扔到海里漂过来的。”

”太好了，太好了，我终于又见到你们了“

从此，小贝壳，还是在海边漂来漂去，只不过它每天都能喝爸爸妈妈在一起玩耍了。

每年的贺岁大片总少不了周星驰的身影，从西游降魔到美人鱼，再到今年的西游伏妖，

每年都能拿个年度票房冠军，尤其是去年的美人鱼，竟然影院排片两个月，这在任何一个在国内上映的

片子来说都是一个特殊待遇，就像建国大业那样的国字头的片子，政府往脸上贴钱，排片没有超过一个月

啊。所以说，周星驰的11亿美人鱼票房绝对是个奇迹。

所以就像以前的过年必看春晚一样，每年过年你不看周星驰的贺岁电影，就觉得这个年没有过一样。

今年过年回来，趁机会去影院看了西游伏妖篇，先不说演员和演技，就是剧情来说，都比不上美人鱼，更不用

说西游降魔了，除了花钱砸特效之外，就是炒冷饭，段小姐的影子始终没有离开过。

先来说一说降魔篇。

一番。

Inconsistent line ending style in svn

发表于 2017-02-17 | 分类于 Mac |

有的时候，SVN 会出来一些莫名其妙的错误，比如这个：

Inconsistent line ending style

有这个问题的出现说明文件的行尾符号不支持。或者说有的行是以 UNIX LF 字符结尾，有的行是以

DOS-style CR+LF 结尾，只要处理了行尾符号问题，svn 也就没有问题了。

有两种方法：

  1. 使用 NotePad++ 打开对应文件，

“Edit”->”EOL Conversion”->”Windows format”

  1. 在文件夹中，选择无法提交的文件，右键 “TortoiseSVN” –> “Properties”. 在弹出框

里找到 “EOL” 属性，删除。

让孩子自己承担责任

发表于 2017-01-17 | 分类于 育儿经 |

正好昨天幼儿园老师还布置了作业，让画灯笼。一是下雨天气，状态不好，二来回来晚了，精神不济。画了两个，就不想画了，我也不知道老师让画几个，就告诉她画好了就收起来。

然后她开始写自己的名字，她总是把头离纸特别近，我就在旁边盯着她，结果她一出神，把名字多画了一笔，我指着纸上的字说，

“这个是什么？”

“我的名字。”

“是不是这样写的？”

“对不起，爸爸，我写错了，我是不小心的。”

然后就伤心的哭起来了，哄了好半天，才哄好。

我就给她讲了一个我小时候写作业的故事，没有认真写，被老师揍了一顿，后来好好写了，老师就再也没有揍我了。

但是，但是她没有get到关键点，哭的更厉害了，

“老师会不会揍我啊？”

最后又哄了好半天，并且答应她，明天早晨送她去幼儿园的时候，告诉老师她的名字不是故意写错的，这样老师就不会揍她了，她才勉强止住哭泣，洗脸睡觉了。

本来我以为这件事就这么完了，

没想到早晨临送她出去上学的时候，她突然对我说，

“爸爸，你今天要记得和老师说两件事！你还记不记得？”

我愣了一下，两件，怎么成两件了？我故意停了下，

“是两件什么事情啊？”

“第一件，你要和老师说，我画上的名字不是故意写错的，我是不小心写错的，

第二件，你要告诉老师，六个灯笼我只画了两个，是因为我太困了”

第一个我知道，第二个她从来没有提起，昨天也没有和我说，我就告诉她说，

”第一个事情，我可以和老师说，但是第二个事情，你昨天没有提前告诉我，而且你画完画还听了会故事，有时间听故事，就该有时间画画，这个我帮不了你，你自己和老师说。”

然后她沉默了一会不说话了。

送她到班里，让她把画拿出来，她拿着画，慢慢的走到老师面前，开始我以为她只是交作业，

”王老师，我昨天只画了两个，是因为我太困了”

”可以，放这里吧”，老师很愉快的答应了，她高兴的过来到我面前，

”爸爸，你还有件事没有和老师说呢，我的已经说过了！”

我又到老师面前，和老师说了一遍名字写错的事情，幼儿园肯定没有那么认真，老师也同意了。

我感觉我和老师说话的时候，比她还紧张。

通过这件事情，我总结了三个问题：

孩子做作业的时候，不要在旁边观看，更不要隔三差五的和她说话或者打断她，让她自己做。有问题了，让她修改，修改不了的，让她想办法解决。

孩子的想法和大人的不同，对同一个事情的看法和角度也不同，有的时候，要蹲下来，从孩子的角度看问题。

孩子的勇气和能力超出我们的想象，不要害怕孩子做不好，给她机会让她自己说话，自己处理矛盾和问题。

我一直到大学还不敢和老师说话，不到万不得已，更不会主动和老师说话，她才五岁就可以自己去和老师沟通，并且自己解决问题，一个大写的服！
