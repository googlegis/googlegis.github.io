Inconsistent line ending style in svn

发表于 2017-02-17 | 分类于 Mac |

有的时候，SVN 会出来一些莫名其妙的错误，比如这个：

Inconsistent line ending style

有这个问题的出现说明文件的行尾符号不支持。或者说有的行是以 UNIX LF 字符结尾，有的行是以

DOS-style CR+LF 结尾，只要处理了行尾符号问题，svn 也就没有问题了。

有两种方法：

  1. 使用 NotePad++ 打开对应文件，

“Edit”->”EOL Conversion”->”Windows format”

  1. 在文件夹中，选择无法提交的文件，右键 “TortoiseSVN” –> “Properties”. 在弹出框

里找到 “EOL” 属性，删除。

让孩子自己承担责任

发表于 2017-01-17 | 分类于 育儿经 |

周三的时候会带娃子上英语班，是在晚上，由于时间紧，放学回来吃了饭马上就要走，有的时候回来晚了就要到晚上九点半才能到家。

去年冬天的某天老师正好还布置了作业，让画灯笼。一是下雨天气，状态不好，二来回来晚了，精神不济。画了两个，就不想画了，我也不知道老师让画几个，就告诉她画好了就收起来。

然后她开始写自己的名字，她总是把头离纸特别近，我就在旁边盯着她，结果她一出神，把名字多画了一笔，我指着纸上的字说，

“这个是什么？”

“我的名字。”

“是不是这样写的？”

“对不起，爸爸，我写错了，我是不小心的。”

然后就伤心的哭起来了，哄了好半天，我又给她讲了一个我小时候写作业的故事，没有认真写，被老师揍了一顿，后来好好写了，老师就再也没有揍我了。

但是，但是她没有get到关键点，哭的更厉害了，

“老师会不会揍我啊？”

最后又哄了好半天，并且答应她，明天早晨送她去幼儿园的时候，告诉老师她的名字不是故意写错的，这样老师就不会揍她了，她才勉强止住哭泣，洗脸睡觉了。

本来我以为这件事就这么完了，

没想到早晨临送她出去上学的时候，她突然对我说，

“爸爸，你今天要记得和老师说两件事！你还记不记得？”

我愣了一下，两件，怎么成两件了？我故意停下来问她，

“是两件什么事情啊？”

“第一件，你要和老师说，我画上的名字不是故意写错的，我是不小心写错的，第二件，你要告诉老师，六个灯笼我只画了两个，是因为我太困了”

第一个我知道，第二个她从来没有提起，昨天也没有和我说，我就告诉她说，

”第一个事情，我可以和老师说，但是第二个事情，你昨天没有提前告诉我，而且你画完后还听了会故事，有时间听故事，就该有时间画画，这个我帮不了你，你自己和老师说。”

然后她沉默了一会不说话了。

送她到班里，让她把画拿出来，她拿着画，慢慢的走到老师面前，开始我以为她只是交作业，

”王老师，我昨天只画了两个，是因为我太困了”

”可以，放这里吧”，老师很愉快的答应了，她高兴的过来到我面前，

”爸爸，你还有件事没有和老师说呢，我的已经说过了！”

我又到老师面前，和老师说了一遍名字写错的事情，幼儿园肯定没有那么认真，老师也同意了。

我感觉我和老师说话的时候，比她还紧张。

通过这件事情，我认为有三点应该是父母要注意的：

孩子做作业的时候，不要在旁边观看，更不要隔三差五的和她说话或者打断她，让她自己做。有问题了，做完了让她统一修改，修改不了的，让她想办法解决。

孩子的想法和大人的不同，对同一个事情的看法和角度也不同，有的时候，要蹲下来，从孩子的角度看问题。
孩子的勇气和能力超出我们的想象，不要害怕孩子做不好，给她机会让她自己说话，自己处理矛盾和问题。

有的时候，孩子的事情就是孩子的事情，家长不要过多参与，她们的能力远比我们想象的要高。家长要敢于让孩子承担责任，敢于放手让孩子去做。

我看长城

发表于 2017-01-01 | 分类于 看电影 |

2016年底的电影有点飘。

前有冯小刚为潘金莲击鼓鸣冤弄巧成拙，再是《长城》被批评为饕餮盛筵，《摆渡人》被批无端脑残，《诺曼蒂克消亡史》被批为不知所云，一个又一个的国产或伪国产电影或这或那的走上了歧途。

潘金莲我没有去看，因为时间不合适，但是书看了，我觉得范冰冰可能演不出李雪莲的那种感觉。《摆渡人》没有去看，因为看预告片就是那种猪肉炖粉条的不需要演技的大杂烩，不是好演员都会接好电影的。有的时候是迫不得已。

《诺曼蒂克消亡史》还没有来得及去看，估计快要下线了，这几个之中，唯有饕餮盛筵我去看了，说实话我其实就是冲着导演张艺谋去的。

从《红高粱》到《大红灯笼高高挂》，从《英雄》到 2008年奥运开幕式，从《满城尽带黄金甲》到《山楂树之恋》再到《金陵十三钗》，我一直都是张艺谋的忠实粉，我觉得在这么多导演中，最富有艺术色彩的就是张艺谋，构思与布局最大胆的也是张艺谋。无论是色彩的运用还是镜头的掠取无一不表现了深厚的导演功底及艺术造诣。

上大学时，正在放《英雄》，我特意用寝室的电脑把各个主角的海报下载下来，拿去彩印出来看。打印处的人问我是不是搞宣传的，我笑笑说不是，我就是喜欢这个片。

看《长城》的预告片的时候，我是完全被熊、虎、鹰、鹿、鹤五色无影禁军给迷住了，我想按照张艺谋的大范围的色彩构思，这五种颜色在无数的怪兽面前肯定要表现出奇妙的构思与对撞。这才是张艺谋的电影。

看完之后，真是觉得没有什么，就是一个完全好莱坞式的爆米花电影。

制作公司应该是万达买的传奇，编剧什么的也是老外，制作完全是好莱坞的模式，估计最终的剪辑也是好莱坞的刀手，着重表现了好莱坞式的个人英雄主义，并安排了一个闪烁的简单爱情线索。

我开始以为请张艺谋过来估计只是挂名，对于万达这种完全不缺钱的金主来说，拉个有名的导演来提高影片的知名度，扩大影片的影响力也是营销的一种手段。后来我才知道，原来张艺谋是乐视的股东，而影片的发行方正是乐视。

对，就是那家疯狂烧钱，疯狂扩张，直到最近暴露出各种问题的乐视。

影片完全不是张艺谋以前的风格，如果按照《英雄》或者奥运会开幕式的节奏拍摄的话，估计会把五色无影禁军的颜色发挥到极致。影片里只着重描述了蓝色的鹤军，那熊军、虎军、鹰军、鹿军都是过来打酱油的吗？

《长城》庞大的中西合璧卡司阵容

亏你的卡司阵容这么牛逼，张涵予、彭于晏、林更新、黄轩，除了殿帅张涵予是女主她义父之外多说了两句话，还和一只饕餮有了亲密接触之外，其它每个人只说了一句台词，可怜的黄轩连台词都没有，那个一说话就让人出戏的皇上还有几句台词呢。人家郑恺还有好几句台词和一个大特写呢！

那些女主背后默默不说话的男人，哪个拉出来不能单独拍一部我在长城打怪兽的大片啊？

其实挺想看张艺谋能拍出一个五色禁军长城大战饕餮的电影的，也就是想想罢了，在这种疯狂圈钱，疯狂扩张，啥啥都能赚钱的时间点，还能有几个愿意按照好电影的模式来拍电影的导演。

对于电影来说，《长城》只能算个及格的影片，60分。也没有网上黑的那么惨。如果能给漫威的傻瓜英雄电影打高分的话，《长城》为什么不能，最起码还有个主题思想。

但是，《长城》仍然算不上一部好电影。

2016年过去了，我很怀念它

发表于 2016-12-31 | 分类于 文章 |

今天是农历正月廿八，过了这个周末，正月就出去了，在出正月之前，貌似老天爷还顺便怀念一下冬季的美好，下了一场雪，可是根本看不到它的存在，只是在温度上又体验了一次冬季，昨天开始，温度又回升了，看天气预报貌似要飙升的趋势，所以，春天的脚步也近了。

过完年之后，就突然感觉到时间特别的紧迫，以前可从来没有这种感觉。

按照中国农历生肖上的说法，今年是鸡年。是我的本命年，红裤衩已经早早的穿上了，红袜子也穿上了，希望能有好的开始。

这是我的第三个本命年了，前两个本命年基本没有感觉，或者说根本没有注意，就嗖的一下飞走了。12岁的时候，还太小，还没有这么的多愁善感；24岁的时候，还在飘，食不果腹，没有精力去多愁善感；等到36岁的时候，突然发现自己老了。

36年过去了，自己随风飘荡，随遇而安。幸好碰到现在的妻子，有了可爱的女儿，稳定了下来，生活也渐渐的丰满起来。有时候觉得自己很走运了，在最坏的时候碰到了最对的人，一个人的一生能有几次这样的机会呢。

回顾2016，回顾这几年，为了生活而奋斗，为了生存而辛苦，有的时候竟忘了身边的家人。

虽然我是个程序员，但是脾气暴躁，对任何人任何事都有着自己过份的要求，对亲人，对妻子，对女儿都想按照自己的意愿走，但往往都是最后伤民劳财。

有的时候，为了生活迫不得已，也会为五斗米折腰，这在从小立志从事文艺事业的我也是不得已而为之。但是为了亲人，为了家庭，为了妻女，为了钱，收敛起自己的戾气，收敛起自尊，收敛起虚荣，老老实实的干活，老老实实的挣钱。在这个看颜值的时代，我只能依靠自己的努力，让家人不至于受到温饱的威胁。

女儿长大了，开朗活波，有的时候，我感觉自己对她要求太严格了，不到5岁的年纪，应该是撒娇耍赖，胡搅蛮缠的年龄，我却用着成人的标准要求她，我是希望她能早点明白成人的世界，成人的规矩，早点明白应该严格要求自己才能适应这个光陆怪离的世界。希望她以后能够越来越懂事，能够明白我的用意。

我以后会慢慢改变豪放派的那种作风，采用婉约派的做法来对待她。

希望她在叛逆期的阶段不会恨我。

工作稳稳当当、波澜不惊，在这家公司工作了6年，没有把工作搞砸，也没有特别突出的成绩，在自己的舒适区又待了三年，有的时候感觉自己在个人成长上是后知后觉型的，总是什么事情经历过失败或者迫在眉睫才会想到要去做，要去改变。有的时候好羡慕那些对自己人生，对自己职业又规划的人，他们在年轻的时候就知道自己要做啥，就朝那个方向努力。

古训有三十而立，四十不惑，但是对于我来说，可能要不惑之后才能而立了。希望在接下来的三年对自己的技术和职业能有个关键性的突破，为了自己能在有生之年能做个引以为傲的事情。

2017 要开始写我的公众号了，以前的那个号，因为不可抗因素，只能放弃了，但是有舍就有得，从一个全新的号开始，也给自己一个全新的开始。

2017 重新整装出发，新的起点，新的期待。

自己顺着碗沿，把边上的白色的沫子喝点，每次都乐此不疲，喝的很开心，很高兴。

可是他女儿却无法理解，无什么不多打一个呢？

我们没有，我们只有不停的干活，在地里干活。年三十大晚上干过，大年初一干过，

正月十五，别人家烟火已经升空，村庄里已经传来阵阵的欢呼雀跃声的时候，我们

还在地里一颗一颗，用手往土里栽菜秧；凌晨3.起来拉车上街卖菜干过，夜里1点

刮大风，为了防止大棚被刮破刮跑，起来收拾大棚也干过；8月份放暑假，中午12点

钻到密不透风的番茄架里铲草，隆冬腊月，冰天雪地，零下14度赤着双手在雪地里刨葱

也干过，回到家的时候，双手已经冻的失去知觉，直接硬邦邦的就靠在煤球炉子上烤。

这样的日子我们都干过，从来，我们没有怨言，没有愤怒，我们只为今天的菜贵了两毛钱，

或者一车蔬菜多卖了5块钱而高兴。

他们坚信，识字就是好，识字就能找到出路，识字以后就不会在隆冬腊月里把冻冰的手靠在

炉子上烤。

我们都走出了那个人虽不多，但是充满了尔虞我诈的村落，在城市里落了脚。

所以我走出来了，虽然一步一个坑，但是坚持走到了现在。

有家有孩子，夏天屋里有空调，冬天屋里有暖气，一天三餐不愁吃穿，现在做的主要追求就是

减肥吃素，锻炼防病。这明明就是20年前过的日子好不好？

所以，人类的物质文明极大的丰富了，然后呢！

10年前，我还认为中国的多级政府贪腐如此，肯定会被推翻，但是现在不一样了，我从骨子里

痛恨这个社会，痛恨这些贪官，痛恨这个体制，痛恨这些人渣。

也许是我反应过激了。

所以我学会了翻墙，架个梯子，自己翻过去，把自己的愤怒隐藏在虚无的英文文字下，

藏起自己的锐气和愤怒，老老实实的养家糊口，

我希望有一天，中国的人都是善良的，我希望有一天，人们都是遵守规则的，我希望有一天，

大家都是友好的，我希望有一天，疾病贫困不再是那些没有劳动能力的群众最大的生存障碍。

我希望政府官员是真正的管人做事的，对普通的群众像对待自己认识的人一样没有偏见和歧视！

很多年前，马丁路德说过，I have a dream! 我也可以有！

努力才会有收获

发表于 2016-09-27 | 分类于 育儿经 |

刻苦努力才能取得成功，有了成功的资本，人生才能更自信！

这本来是一个通俗易懂，童叟无欺，放之四海而皆准的道理。但有的时候，不但孩

童做不到，就是大人也做不到！！！

上周四，上幼儿园中班的娃娃回来说，老师让家长拿着小孩的手画五角星。所以我

先拿了张纸告诉她怎么画，她学的很快，练习了5-6个之后，她就画的有模有样了，

虽然没有达到横平竖直，角度相同，但从形状和笔划能够看出五角星的样式和工整

了。

所以很满意，就夸奖她画的认真，而且很好看！ 受到夸奖的小孩子很容易表现的，

然后自己又马不停蹄的画了很多个，每个画的都很不错！每画一个就拿过来让看是

否画的好，意思就是你再表扬我一下呗！！！

连着画了几个工整五角星之后，我说帮你找张纸，你自己画，然后明天交给老师。

前面都画的那么好，我就没有看着她画，她还是每次画一个，然后再过来让看，

前面两个画的还不错，可是后面的画的越来越差，不但笔划顺序错了，连基本的

直线都画不好了，基本就像是一个手枪了。五角星能画成手枪，也算是潦草到家

了。

她画了6个拿过来的时候，我看后面几个已经一塌糊涂了，所以假装生气的说，

这几个肯定是你没有用心画，你刚才自己画着玩的时候都能画好，现在却画成

这样，说明你没有用心认真的画，我再给你重新拿张纸，再重新画。

她很委屈，可是自己画的确实不好，只好再画，我停留了一小会去看她，

第一个就画成了手枪，我严厉的告诉她，这样肯定是不对的，你是不是忘了怎

么画了，我又找来一张纸，和她一起画，我画一个，让她看着，然后让她自己

再画一个，我再画一个，让她看着，然后再让她画一个。等她学的像模像样了，

就让她自己画。练习了4-5个之后，让她再重新画了一张，总共画了7个，前面

4个画的还可以，后面3个的线条稍微有点弯曲，假装生气的稍微指点了一下，

放到书包里让她明天带到学校交给老师！！！

我本来只是想锻炼她的耐心，注意力和认真程度，没有想到却带给她莫大的自信。

第二天，老师让小朋友自己画五角星，她自己在幼儿园里画的又快又好，很快就

完成了，还受到了老师的表扬。然后老师告诉她说，去看一下哪个小朋友没有画

完或者不会画，你去帮她画，据她自己说，她在班里问，谁需要帮忙画五角星？

然后有很多小朋友举手，她就帮了好多小朋友画五角星。当然了还是受到了老师

的表扬。

下午从学校练完舞蹈回来，回来后，她兴高采烈的绘声绘色的和我描述了一番，

休息了一下，就说要学习，结果那天晚上停电了，刚开始的时候没有找到蜡烛，

所以一直告诉她等来电了再学习，最后捣鼓出来一小截蜡烛，刚点亮，她就找

来一张纸，说是要画五角星。

哈哈！！！ 家里每回来一个人，她都要拿着画的五角星让家人评价她画的是否好。

言语中充满了自信和激情。这也是我想要的。

鼓励+表扬–>努力+成功–>自信

看来在教育孩童的这条路上，我也是边学习，边摸索，以后的路还很长！
