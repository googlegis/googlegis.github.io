添加系统引用和包的引用

发表于 2016-07-08 | 分类于 Xamarin |

本文接着讲如何添加 dll 库和引用包，开始的时候，我看项目里有

refrence 和 packages 两个部分，搞不清两者关系，后来从添加的内容看，

refrence 中添加的都是和 .Net 有关的 dll 库，packages 里添加的是

和网页开发相关的 js 库(当然也包括一部分和后台 C# 相关的 dll 文件)。

后面一篇专门讲 log4net 的文章时会有提到！

要添加系统库，需要在 Refrences 文件夹上右键，选择 Edit Refrences…

或者直接双击 Refrences 文件夹。

然后选择自己需要的库，这里添加最常见的 System.Data 库，

因为你要保存查询出来的数据库结果集，这个使用非常方便。 打勾-OK！

然后就可以看到 Refrences 中的 System.Data 引用。

在.cs 文件中，添加 using 。然后在代码中就可以直接用 DataTable class

进行代码处理了。

添加包的引用和 Refrences 差不多，文件夹上右键，Add Packages… 或者双击，

然后弹出添加引用的窗口，这里是使用官方 NuGet Gallery。 如果你够仔细的话，可以

在配置中找到包源的配置界面，可以自己添加包源配置。

比如，我们搜索 jQuery 。会出来 jQuery 库， 和 jQuery UI 库，我们选中

jQuery UI 库， 能够看到在右侧的说明中，有 Dependencies 依赖， jQuery(≥1.6)。

直接勾选 jQuery UI，点击 Add Package，然后能够看到正在安装的过程。

安装好之后，就能在 Packages 中看到 jQuery 和 jQuery UI。

这说明不需要担心你选择的库的依赖，直接点击安装，其它的事情，IDE 来帮你做。

然后我们打开项目中的 packages.config 文件，来看一下我们项目中 yij

引用的类库和包！

打开 Scripts 文件夹，能够看到引用 jQuery 的文件都在这里了。

这里需要严重吐槽一下，这种依赖添加只会按照最小版本进行添加，这里添加

进来的 jQuery 的版本就是 1.6.4 ，所以如果想要使用更新版本的 jQuery，

还需要自己手动添加。

但是还有另外一点支持不好的地方，每个库都只能引用 NuGet 中最新的版本，

比如这里的 jQuery 库是 1.6.4 版本，太旧了，想更新到新版本，就会直接更新

到最新版本，而现在 NuGet 上最新的版本是 3.0.0, 而且其它依赖 jQuery 的

库的版本还没有来得及更新到 3.0.0 的版本，所以这里就又是一个坑。

需要自己手动下载 jQuery 库，进行添加引用。 我想如果添加库时，能够有一个

版本选择的功能的话，是不是方便很多。

打开命令行工具：

jays-MacBook-Pro:~ jay$ vim ~/.bash_profile

把下面这行添加到 .bash_profile 文件的后面，

export PATH=/usr/local/git/bin:$PATH

保存文件，退出编辑！

再验证版本，就会看到最新的升级版本号。

我看小王子

是的，每个人心里都有一个小王子！

我和老婆刚认识的时候她就给我推荐过小王子，可惜为一直没有放在心上，也一直没有特意的看书，于是趁着电影上映的时候陪着老婆和娃子去看了一遍。

看完之后，确实被感动到了，应该说：每个人心里都有一个小王子。

有时候，我们是长大了，但是长大的过程中最可怕的不是年龄的增长，而是遗忘。很多时候，我们不是因为长大了而变得功名利禄，而是我们把有些东西遗忘了。忘在一个角落里再也想不起来了。

如同小王子的玫瑰，小王子曾经是那么的深刻的爱着，可是当小女孩去找王子的时候，他还是想不起来玫瑰是什么了！

那我们是否为了生存，为了面子，为了成长，为了金钱，为了工作，而忘记自己也曾经有过一朵玫瑰呢？

其实，一直到现在，我都还不知道作者的名字，看电影的过程中我想，小王子的作者该有多孤单啊！

想在成人的社会规则中保持自己的童心，想在成长之后保持自己的玫瑰，可是他不能被社会所接受。被听到掌声就脱帽致谢的自大狂鄙视，被自以为是高高在上的电梯员鄙视，被唯利是图的老板鄙视。

他不能被世人理解，也许当初曾努力的保持自己的童心，可是发现在纷扰的尘世中，根本无法继续生存下去。所以他接受了所谓的帮助和教育。

那些人把他教育成了现实中需要的人，一个不停打扫烟囱的好员工-王子！

他忘记了自己是谁，忘记了小狐狸是谁，忘记了玫瑰是谁，甚至忘记了星星，忘记了绵羊。

那些可怕的大人！

有时候，我会想，也许作者本人就是那个飞行员老头，童年里有着飞行员的梦想，还喜欢每天在夜晚里看星星，或许他在乡下或农场里真的有一个狐狸好朋友，也许他第一次看到玫瑰花的时候，真的就深深的爱上了她，可是玫瑰刺伤了它，也许小的时候幼儿园的老师让他画绵羊的时候他就真的画了一个盒子，也许他曾看着天空中飞过的鸽子，幻想过让鸽子带着他飞过天空，飞到很远的星球，他甚至都给好多好多的其它星球起了名字和编号，其中一个就是长满猴面包树的星球，还有他的玫瑰也在那个星球上，还要不停的把猴面包树锄掉，不然就会变得很糟糕。

他的脑子里长满了各种各样奇怪的想法。

可是他必须要长大，他也要上学，也许他上的就是沃茨学院，学校把他按照标准规格制造成了合格的的大人，他有工作，有朋友，有生活。只是他没有了童年。

他也许曾经试图保持自己的纯真，可是一丝不苟的警察，自大狂，电梯员，老板和生活一次又一次的把他打回了现实。

他必须保持不停的工作，对于大人来说工作就是一切，只有工作才能获得大人的认同。每当一个小王子长成大人时，他的星星就被封闭在那个罐子里了，不但丢失了童年的一切，也丢失了找回童年的信心。

在大人的世界里，会不停的有人提醒你：你是一个大人，做你该做的！

是的我们必须要做我们该做的，不管为了什么，我们都应该去做。但是在成人的世界里，我们也许可以偷偷的保留一点童心。

它可以时刻提醒我们，在面对孩子那些莫名其妙的想法，和不合规矩的举动时，应该蹲下来和孩子说话，而不是高高在上的怒吼，让孩子仰望你生气愤怒的脸庞。

孩子仰望的应该是浩瀚的星空。

小王子也许每两年就该看一次，也许每次看的心情和体会都不同。

娃子现在五岁了，再过两年就该上小学了，那时她就有自己的玩伴和更多的想法了，目前来说她还属于天真烂漫的年龄，还没有作业考试的烦恼，还没有青春躁动的无处安放，更没有安家立业的压力，这是她最无忧无虑的时间，也许现在是她最接近小王子的状态。我希望能在她最好的季节里陪她经历生活中的每个细节，知道她每次的喜怒哀乐，陪着她渐渐长大，保护好她的小王子。

我的小王子已经丢了，再也找不回来了。

因为我是大人了！
