西游4-八戒

发表于 2017-03-19 分类于 谈天说地

肥头大耳猪托生

天河元帅下凡尘

九齿钉耙浑不怕

在此应等有缘人

广寒仙子月宫住

酒醉不期动凡心

触犯戒条天威怒

云栈洞里暂留身

食色贪惰穿肠过

菩萨命等取经人

皈依我佛念八戒

西天路上踏晓晨

西游3-吾净

天上卷帘将
失手碎灯盏
贬入流沙河
万剑刺心穿

寸草不能存
鹅毛入水渊
食肉饮血妖
本是位仙班

十世取经僧
只为此生缘
一朝归空门
拜佛向西天

西游2-前世

发表于 2017-03-17 分类于 谈天说地

江河东流水滔滔

万年身世谁知晓

本是名门宦臣后

静卧佛寺木鱼敲

佛祖座下金蝉子

因果轮回到今朝

观音引渡佛前光

万里取经路迢迢

晨前牵马踏蹬去

两界山外多魔妖

唐王驾下求经僧

西天拜佛路途遥

西游1-大圣

发表于 2017-03-16 分类于 谈天说地

头戴紫金冠

身披金缕衣

家住花果山

脚踏五彩云

天父地母生

日月孕我身

擎天神针铁

来去筋斗云

湖海纵广阔

难容不平心

戏游蟠桃宴

力战百万兵

如来佛祖至

五行压我身

吃雪饮雨露

盼来取经人

佑护唐三藏

遁入佛家门

万念皆抛去

求修正果身
