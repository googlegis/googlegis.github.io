在Windows下安装Node.js

发表于 2015-05-13 分类于 Translation

JavaScipt 将要成为Web开发者的首选编程语言，前端开发人员使用JavaScript加强用户体会及交互操作，后台web开发使用ajax调用web services。服务端web开发人员也开始使用javascript，主要是因为它的事件驱动及非模块特性带来的高速度、高性能。

事实上，掌握一门简单的语言，就能开发一个全栈web程序，javascript就是这样一个机遇。关键就在于Node.js，作为Chrome V8 JavaScript运行引擎，它让在服务端运行javascript成为了可能。

Node.js 也被用来开发桌面程序，还可以作为部署工具使得开发web站点更加容易。比如：在你的桌面电脑上安装Node.js,能够快速转换CoffeeScript 到JavaScript，SASS到CSS，并且能大幅减少HTML、JavaScript、图片文件的大小。使用NPM(安装和管理Node模块的工具)能够在你的开发工具快速添加各种有用的工具。

前面一篇文章《在mac下安装Node.js》中，我讲述了如何在windows下安装Node.js,同样作为姊妹篇，我这篇文章讲如何在Windows下安装，幸运的是，对于windows用户，安装过程要比在mac下安装简单的多。

安装简述

使用Node.js官网的安装包进行安装Node 和NPM比较简单。

安装步骤

  1. 从官网直接下载安装包，下载地址：http://nodejs.org/download/.
  2. 点击下载的安装包进行安装。(就是第1步下载的.msi文件)。
  3. 按照默认的选项和步骤，一路next下去。
  4. 重启。
