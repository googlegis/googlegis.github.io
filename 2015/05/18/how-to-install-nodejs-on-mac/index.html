在Mac上安装Nodejs

نُشر في 2015-05-18 عُدل في 2017-05-25 في Translation

node.js 作为常规解释javascript浏览器的替代品，允许你以命令行方式解释javascript，而且在web开发方面已经成为了一个现代化的工作流。使用

node.js 的管理包工具NPM能够对nodejs进行各种工具包的管理，其中相当多web开发重要包里的一个就是grunt.js

想要在OSX10.10 Yosemite中安装node.js, 你可以下载二进制预编译包，这种方法比较方便而友好。打开http://nodejs.org/,

点击安装，下载最新的安装包。

打开下载的.dmg 文件，按照提示安装的话，node和npm都会自动安装。npm 就是 Node Package Manager,其实就是nodejs的各种库的安装管理工具。

安装完成后确保 /usr/local/bin 在你的path路径下，打开命令行工具输入：

如果没有存在，请在home目录下的.bash_profile 或者 .bashrc 中任选一个添加。

安装完成后，检测是否安装正确。打开命令行工具，输入 node，开始javascript命令模式。

退出nodejs的编辑模式，按两次‘control+c’即可。

如果你的机器已经安装了较早版本的nodejs,在安装新版的时候会直接替换旧版本。
也可以在命令行中检测node 版本
