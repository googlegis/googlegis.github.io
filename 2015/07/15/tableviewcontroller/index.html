Swift 2.0 纯学习之旅--7.tableviewController

发表于 2015-07-15 | 分类于 Swift | | 
这里是学习 iOS Apprentice 的第二课 Checklist 的笔记记录，仅供自己查阅。

tableView 和 nagView 是 iOS 开发中最常用到的两个 viewController，不论你的主界面就是

tableView 的列表显示，还是仅仅用 tableView 来显示属性设置，它都是一个普遍而又重要的

组件。

在 Checklist 的开发过程就大量使用了 tableView 和 navView，navView 是从一个 scence

跳转到另外一个 scence 的重要纽带，这两者结合在一起，恐怕缺少的就只是你的想象力了。

everything is possible here!

新建项目

打开 Xcode ，新建 Single View Application ，

在 TARGETS 选中 Checklists，然后在 General–>Deployment Info–>Device Orientation

中确保只有 Portrait 被勾选。

到这里一切似乎都很熟悉，就是把 storyboard 中的 viewcontroller 与代码中的 swift 文件

绑定， 其实可以不是同一个名称，也可以是不同的名称，同一个名称是为了更好的同其他的

viewController 区分开来，因为我们在这个项目里将会不只一个 viewController 。

然后在 Attributes inspector 中，勾选 Is Initial View Controller , 使其作为默认的

再次选择 Table View Cell ，然后在 Attributes inspector 中，选择 Accessory 为
