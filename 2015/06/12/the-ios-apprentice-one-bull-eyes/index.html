  1. Disable SizeClass

从iOS8开始，Xcode提供了一个 size classes 或者叫做 universal storyboards的

功能，能够让你一次设计，处处运行，也就是你在这里一次性设计完成，就不论它是iPad还是iPhone

还是iPod，不论它是 3.5-4-4.7-5.5-8.9 英寸了。但是这里我们先不讨论，我们要把该功能关闭。

左侧导航栏选中 Main.storyboard, 右侧点击 File inspector，取消勾选的 Use Size Classes。

当Xcode提示你是否确认时，点击 Disable Size Classes 继续，你会看到你的画布大小变为了实际的

的iPhone大小。

然后运行程序测试。

－－－－

如果使用了Size Classes，因为是要面对所有尺寸的设备，所以需要对button的相对位置重新
定义，而不是简单放置在中间就可以了，在Size Classes 模式下， 你手动放在中间，运行的
时候并不在中间；而取消掉了Size Classes模式，你放在哪里就出现在哪里，因为只需要对一种
类型设备负责。

6.landscape 模式

在Main.storyboard 中选中 View Controller，

或者在 Outline pane中，选中View Controller，

然后在 show Attributes Inspector 中的 simulated Metrics节点中，将 Orientation 中的

Inferred 改变为 landscape。

然后运行试试，没有变？没有变就对了。还要改个地方。左侧导航栏选中项目，选择 General，然后在

Deployment info 中的 Device Orientation 中，取消勾选 Portrait。 只留 Landscape Left

和Landscape Right。

再运行测试，OK。
