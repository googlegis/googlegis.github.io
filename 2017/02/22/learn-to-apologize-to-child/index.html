学会向孩子道歉

post.posted 2017-02-22 | post.in 育儿经 | | post.viewssymbol.colon

有的时候，你必须要蹲下来和孩子道歉!

今天早晨送娃子去幼儿园，刚到那里正准备放书包，班里的王老师，特意走到她身边说，

“我要和你说件事情，老师昨天家里有事了，忘记把画画的照片发给爸爸了，今天放学发好不好？”

小娃娃想了一下，

“好！”

这时，我才想起昨天下午放学接她的时候，发生的一件事情。

昨天放学接她，小娃子很兴奋，告诉我说，因为昨天画的画好，老师奖励给她糖吃，还有口袋里有好多其他的好东西。

她问我，

“老师有没有把我画的画发给你看？”

我说没有啊。

她背着小书包，蹬蹬蹬的跑到王老师面前问，

”王老师，你有没有把我画的画发给爸爸看？”

王老师笑着说，

”还没有，今天放学回家后发好不好？”

小娃子同意了，然后和老师再见回家了，回到家又和家里人说了一遍画画的事情。

估计昨天老师太忙忘记了，所以就有了今天早晨老师给学生道歉的一幕。

其实我是很感动的，今天早晨去的时候，教室里已经来了很多同学，王老师还正在和园长说话，看到小娃子进来，王老师还特意过来和她道歉，说明原因并达成一致意见。

我觉得这是一个伟大的老师。

其实对于每个家庭来说，自己的孩子就是全部，是100%；但是对老师来说，一个家庭的孩子，在30多人的班级里，只能算是3%。对于老师能记住和她说过的话，记住了答应她的事情，并因为没有做到而专门到她面前商量道歉，能够像家长一样对待孩子，我很感恩。

不知道有没有家长从来没有和孩子道过歉，自己做错了也是做错了，但是对孩子没有任何表示，不但没有‘我错了’这三个字，甚至连一个交流的机会都不给孩子。

有人认为，孩子是我生的，长这么大是我养的，钱是我出的，我给孩子道什么歉啊？

还有的人会认为，小屁孩一个，什么都不懂，道歉也没有意思。

其实，这些想法都是有偏见的，现在的小孩发育快，思想成熟早，在很小的时候，就有自己的想法了。

家长如何和孩子建立平等的关系，如果在孩子面前建立威信，全都是通过平时生活的点滴相处来建立的。

你可以对孩子不道歉，可以对孩子大声恐吓，可以说话不算，可以辱骂甚至打骂家人，可以熬夜睡懒觉，可以好吃懒做，可以抽烟酗酒，可以脏话连篇，这些都可以做，但是孩子也会学着做。

有的家长抱怨孩子打滚撒泼，有的家长抱怨孩子胡搅蛮缠，有的家长抱怨孩子磨蹭拖拉，但其实你在与孩子的交往是否已经教给了她这些行为准则了呢？是否也做到了令行禁止，今日事今日毕呢？是否蹲下来平等的与她对话，让她知道做好的结果，与做坏的结果的不同呢？

其实家长的一言一行影响着孩子的社会准则，家长的一举一动影响着孩子的生活习惯，这些都在孩子幼小的时候，深深的印在脑海里，长时间的习性的积累，就会对孩子造成不同的影响。

父母只有管理好自己，才能管理好孩子。

不以恶小而为之，不以善小而不为。
