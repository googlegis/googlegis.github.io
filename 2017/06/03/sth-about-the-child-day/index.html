儿童节演出记事

发表于 2017-06-03 | 分类于 育儿经 | |

娃子幼儿园的演出是安排在六月二号，星期五。

\<– read more –>

老师们早早就排练起来，家长们也都提前该请假的请假，该调休的调休，就等这天和孩子们一起开心的玩一天呢。

按照幼儿园安排，演出是租用了其它学校的礼堂，早晨8点开始，因为还要给孩子们化妆，所以统一要求7点左右，由家长带领孩子在演出礼堂集合。
演出过程相当顺利，中二班的小朋友参与演出的几个节目演出效果和观众反响都很好。一个是环保时装秀，一个是儿童剧《迷路的小蚂蚁》，一个是中班舞蹈班的《粒粒皆辛苦》。
感觉老师们真的是太厉害了，那些平时活蹦乱跳的，一刻都不会老实安静坐下的小朋友，竟然也能踩着音乐的步伐，扭动青春的小腰，跳出鲜艳的色彩。

其实，在这次活动中，无论是从准备工作，舞台表演，团队合作各个方面，娃子还是学到了一些东西的。

特殊处理

因为六一演出是特殊的事情，对于家长特殊，对于孩子也特殊，平时总能睡到7点才起床的娃子，这次6点就要起床了，所以前一天放学前，老师就给小朋友们通通气，晚上8.30必须上床睡觉，在家里的时候，也对孩子特别交待，明天的演出和平时在学校跳舞可不一样，我们要比平时早起，就是平时你刚醒的时候，我们就要到演出的地方了。首先在孩子心里一个暗示，一是为了提前预防，二是为了万一明天出现哭闹的情况，能够站住脚和她讲理。

集体荣誉

娃子在演出的时候，相当卖力，而且对演出相当上心，平时在家没事的时候，就表演她演的蚂蚁妈妈，不但表演蚂蚁妈妈，还和我们讲出场顺序，和别的角色的表演和台词。在这次幼儿园演出的过程中，碰到几次有的小朋友不对的地方，她就做些小动作提示同伴，她的这些举动还是出乎我的意料的。回来我问她，你有的时候，在台上向小伙伴挥手让她们走动，是老师安排的还是你自己想做的啊？她回答说，是自己想做的。于是就趁机和她聊起了集体的成功和荣誉的含义。对于班级表演或者集体活动，需要大家都能完成，而且表现的都好才能算是好。不然的话，就算你一个人变现的好，也只是你一个人觉得好，在大家看来，你们一起表演的节目就是不好，所以在集体中，能帮助同伴就尽量帮助，因为你们是一个集体的。

安稳军心

确保在演出前和演出后能和孩子见上面，沟通一下。这里的见面不仅仅是吃喝拉撒的问题，你可以和孩子聊聊环境，聊聊心理，聊聊其它小朋友的表演，主要是给孩子以安慰和放松心情，防止因为环境或者其它原因而突然紧张，造成临场发挥失误。娃子回来的时候，妈妈夸她表演的时候，跳的不错，她说，
‘其实，我一开始还是有点紧张的。’
‘那你什么时候开始紧张的呢？’
‘就我一上去看是一个大舞台的时候’
‘那你什么时候起不紧张的呢？’
‘我看到爸爸妈妈都坐在下面的时候，我就不紧张了，后来我就跳出了能量！’

突发事件

其中有一个班表演节目，其它的小朋友都开始跳舞了，另外两个小朋友才上台来，看那匆匆而来的样子，应该是被老师强制上来的，其中一个孩子很快找到了位置，融入到表演当中，但是另外一个孩子，心理素质不是很好，一上台来，发现位置被占了，而其他的小朋友都正在跳，面对这种突发状况，这个小朋友一下懵圈了，有点不知所措。我看到指导老师，在最前面几次给她手势，让她站队或者跟着做，看她哭了，还给她大拇指鼓励她勇敢起来跟着做，但是这一切都是枉然，小朋友根本注意不到指导老师，而指导老师因为还要顾及其它的小朋友，不能亲自跑上舞台进行安排，所以那个小朋友从上台一直哭到结束谢幕，然后跟着下去了。所以，孩子的世界，并不总是风平浪静，波澜不惊的，她们有的时候也要面对困难，面对困惑，面对阻力，面对竞争，甚至会面对突发事件，培养孩子的心理素质，将会是孩子成长路上的一块重要的垫脚石，而这种素质的培养是越早越好。

情商

情商，对幼儿园的孩子同样重要。对于很多家长来说，都希望自己的孩子能出演重要角色，能站在舞台上表演，但是对于老师来说，一个班级太多的学生，不可能完全照顾的到，所以无论在哪个节目里都有主有次。这个时候，孩子的情商就决定了她的角色。有可能她不是最漂亮的，也有可能她跳舞的功底不是最出色的，但她一定要是最能领会老师意图，最能表达出角色含义的。出演迷路的小蚂蚁那个小女孩，看到旁边的一个代表鲜花的牌子倒了，她趁机把牌子扶起来了，这肯定不是老师教的，也不是家长嘱咐的，而是她自己想到的，觉得应该把它扶起来，这样的话，不但给自己的节目加分，她肯定会在老师的心目中也会加分的。类似的举动肯定也能在平时的学习或者排练中表现出来，这也许是她为什么能演主角的原因。

正所谓，世事洞明皆学问，人情练达即文章。在生活中的任何一个小事，只要能认真思考，都能作为人生的参考和指导。有些道理并不是写在书本里或者在老师的作业里，就是在普通的生活琐事里，家长要善于发现，善于引导，用我们身边的小事来教育孩子，肯定效果更好。
