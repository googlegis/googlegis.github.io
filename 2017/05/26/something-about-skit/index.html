裙子风波

发表于 2017-05-26 | 分类于 育儿经 | |

在五六岁的小女孩群体中，正在掀起一股爱莎风，就是那个冰雪奇缘，唱着 let it go ,一跺脚起来一座冰雪城堡的公主，但很多倒对安娜公主不怎么喜爱，有次我问娃子，你问什么喜欢爱莎，娃子回答说爱莎会魔法，会变出雪花。

现在，无论是买衣服，买鞋子，买书，买牙刷，买头饰，只要是和冰雪奇缘和爱莎沾上边，总是兴奋的能唱起 let it go。

目前为止，已经给她买了两套冰雪奇缘的裙子了，一套是圣诞节英语班里搞活动，给她买了一套长款作为圣诞礼物送给她的，里面包括皇冠，魔法棒、发辫、披肩一整套服装，花了不少银子，第二套是六一前给她刚买的，是一套短款的适合夏天穿的，有给她买了一双公主鞋，每天穿着爱纱裙，穿着公主鞋，在屋里跑来跑去的到处唱，装作会魔法。

给她买第二套裙子还有个故事。

以前那套长款，深得她心，不但上学的时候穿，周末在家的时候也穿。但最近天热的实在厉害，几次和她商量要她换成其他的裙子穿，她都不愿意，再热的天她也要穿着长裙到处跑，每次身上都有很多汗，碰到天热的时候，就算在家里坐着，脊背后面也会不停的有汗出来。所以我曾经试过几次，严禁她穿爱莎裙，有的时候，她几近着魔似的，晚上睡觉前，会问我，

‘爸爸，我明天穿冰雪奇缘的裙子吧？’

早晨醒来，也会问我，

‘爸爸，我今天穿爱冰雪奇缘的裙子吧？’

有的时候，我会故意的禁止她穿，有的时候，我会告诉她，如果你表现的好，我就会让你穿。

所以，为了能够穿上爱莎裙，她会尽力的表现的去做好每一件事。

她知道我是很严厉的，而妈妈则是很好说话沟通的，所以，有的时候，她睡觉前，会告诉我，

‘爸爸，我明天早晨换衣服的时候，你不要在屋里好不好？’

因为我不在屋里，她就可以讨好妈妈，让妈妈允许她穿爱莎裙。

有天晚上，她问了我很多次第二天能否穿爱莎裙，等到睡觉前的时候，她又问了一次，我告诉她说，

‘本来，你只要表现的好，把你自己该做的事情做好，即使你不说要穿爱莎裙，我也是可以让你穿的。但是，你每天把所有的心思都放在要穿爱莎群上，每天都在不停的问，我觉得你这样做是不对的，所以你明天不能穿，而且这一个星期都不能穿，当时不再和我提穿爱莎裙的时候，我倒有可能帮你找衣服的时候，把爱莎裙拿给你穿。’

第二天早晨，她趁我不在屋里，自己把裙子给穿上了，然后一边高兴，一边忐忑的从我面前走过，我看见了，问她，

‘昨天晚上，怎么可以和你说的，今天可以穿爱莎裙吗？’

‘昨天说的可以啊！’ 她准备狡辩

我再次厉声问她，

‘我昨天怎么说的？你可以穿吗？’

‘不可以！’

‘那你为什么穿了？去把她换掉穿T恤！’

她委屈着，去把衣服换掉了。整个早晨直到送去学校，也不怎么开心，也不和我怎么说话。

老婆在网上问我，是不是可以多买几件裙子，让她随便穿，这样每天她都能穿裙子，是不是就对爱莎裙没有那么顾恋了。我告诉老婆，其实我让她换掉的主要原因，不是因为她总是穿爱莎裙，而是因为我昨天已经明确告诉她不能穿了，而且她早晨不但穿了，而且还说谎。如果她没有说谎，我倒可能看她已经穿上而不会让她再换掉，说谎是不能原谅的。而且她为了穿爱莎裙，连冷热都不顾了，平时在家就是在镜子面前照，连放学后看书都不是很安心了。所以，有的时候，我会让她换掉衣服，或者命令她一个星期不能穿爱莎裙。但其实从来没有严格执行过，最多两天不穿，第三天就会让她穿的。

再后来，终于要过六一了，老婆特意从网上买了一款短袖的适合夏天穿的爱莎裙，她看到的时候，高兴的都叫了起来。

现在她又两个爱莎裙了， 那个长款的因为太热不能穿了，这个短款的更是爱不释手。

六一儿童节学校表演，老师又发了一套黑色的 T恤加裙子，又是爱不释手的穿了起来，表演回来那天就要穿上出去显摆。我告诉她说，

‘这个是新买的衣服，还没有来得及洗，而且你今天表演跳舞的时候，上面出了很多汗，还有好多灰，现在是肯定不能穿，如果你想明天穿的话，最好趁现在有太阳，赶紧洗干净了挂起来晒着，这么明天你就能穿了。不然的话，你明天可能都穿不了。’

她听了很紧张，赶紧把手里的东西放下，自己找来盆，接上水，把衣服泡进去。

‘爸爸，你告诉我怎么洗好不好？’

‘好啊！’

我告诉了她哪几个地方是重点注意的，如何揉搓干净，如何把衣服漂洗干净，等她洗好，我帮她把衣服挂起来晾着。

等到第二天出去玩的时候，她终于可以穿着自己的裙子出去玩了。
