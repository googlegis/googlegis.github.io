做好家长的那部分

发表于 2017-05-24 | 分类于 育儿经 | |

上周一，去接娃子放学的时候，听到有的家长问老师有没有作业，老师说没有。回到家也就放心大胆的让孩子玩耍起来。

晚上 8:30，准时要求她去刷牙洗脸，然后上床讲故事、睡觉，等这一套动作打完之后，已经 9:30 了。

老婆下班回来问,

‘娃子的今天的作业做了没？’

‘什么作业？’

‘老师发的校讯通，里面有个剪纸的作业。’

我拿出手机看，果然在8.20左右发的消息，我看了即将睡熟的娃子，

‘算了，明天再说吧。’

第二天早晨起来，我问娃子，

‘你们老师昨天布置作业了吗？’

‘老师说没有’

我相信她不会对我撒谎，我就告诉她说，

‘昨天你准备睡觉的时候，老师发了校讯通，让做家庭作业，是剪人的头像，并剪出五官，爸爸看你睡着了，就没有叫你，等会到学校了，我会和老师说，告诉老师昨天发校讯通的时候，你已经开始洗漱睡觉了，今天放学再做这个作业，好不好！’

一开始的时候，娃子听说有作业，她的脸色还是很严肃的，因为我以前告诉过她，如果是她的原因忘记做作业的话，要自己去和老师解释的。后来听我说，我去和老师解释的时候，脸上都要笑出花来了，高兴的说，

‘好啊！’

到学校之后，我和老师解释了一下情况，老师也同意了，然后和娃子再见就回来了。

有的时候，并不一定什么都让孩子来做就是好的。以前我写了一篇文章，《让孩子承担自己的责任》，写娃子自己忘记做作业，然后让她自己去和老师解释原因的情况。

要不要让孩子来承担责任，我觉得还是要分情况的。孩子的责任，就是孩子的责任；家长的责任，就是家长的责任，要区分开来，我主动把家庭作业这个信息拦截下来了，没有告诉她，这就是我的责任，我就要负责和老师解释。

当时，老婆说有作业的时候，我还是犹豫了一下的，要不要把她叫起来做作业呢？如果叫起来做的话，娃子应该也会乖乖的做完再去睡觉，而娃子也基本不会反对什么的。但是可能对娃子来说就会造成一种疑惑，让刷牙洗脸睡觉的是你，我都要睡着了，把我叫醒起来做作业的也是你，到底想让我怎样呢？

有的时候，有的事情，家长一句话就能解决的事情，还是不要把责任揽到孩子身上，毕竟很多时候，孩子的内心还是比较敏感和脆弱的，她们有自己的尊严和信仰，有自己的理解和观念，只是很多时候并没有被大人承认和赞同而已。我们家长要做的就是要把孩子不正确的观念和想法引导过来，好的积极的想法更加强化和增强，在必要的时候，伸出胳膊帮孩子挡一箭，还是必要的，反正又不需要刮骨疗伤！
