养娃不易，且打且珍惜

发表于 2017-11-07 分类于 育儿经 Disqus：
今天的你是哪种情绪？

我，脾气越来越差了，越发的难以控制住自己随时都会爆炸的脾气。

有时面对熊孩子的无理取闹，心中总有一股无名之火，腾然而起，感觉自己的小宇宙就要爆炸了，脑浆在那里咕嘟咕嘟的冒泡，当达到血脉贲张，喷薄而出的时候，便会伸出处于加速运动的巴掌。

星期天的时候，带娃出去玩，中午玩的比较晚，就在外面吃了她最喜欢的KFC，回到家，我告诉她去小便然后睡午觉，她在那里左看看，右站站，比划比划裙子，翻上两页书。足足磨蹭了有半个小时才去开始做。当我再次严厉的警告她时，她反抗我说，

“我不是正在做吗？”

“我让你干嘛？”

“你让我小便啊！”

“那你在干嘛？”

“我在小便啊！”

“那你刚才在干嘛？”

“小便啊！”

终于按捺不住心中的怒火，一耳巴子拍在了胳膊上，我让你小便，

“从我开始叫你小便，到你现在开始小便，花了40分钟，你左蹭蹭，右蹭蹭，就是不好好做，说你两句你还不承认”

“啪啪！” 又是两耳巴子打在了身上，

娃子终于忍不住，哭了起来。

这只是漫长的养娃史中的一个掠影，相信每个父母都有过因孩子的无理取闹而怒不可遏的情形，她可能没有按照你说的方式，去刷牙，去洗脸，去吃饭，去穿衣，去扎头，去睡觉，去叫人，去礼貌，去玩耍，去回家，去上学，去读书，去做作业，去考试，去做任何你认为应该做，但是却没有做到的事情。

每打一次，自己都自觉反思一次，反复告诉自己，小孩，小孩，总归是小孩，如果孩子能知道这些，就不能称为小孩了。

有的时候，孩子的的错误，其实并不总是孩子的错，只是大人们觉得事情脱离了自己的控制，没有按照大人的期望的节奏发展，而出离愤怒。

在当时的我看来，花时间陪你到处玩，花钱让你好吃好喝，结果，让你去小便睡觉都这么难，说你两句就顶嘴，看老子打不死你。

是的，这个时候，我的心里想的只是老子，那个已经死了上千年的老子。

愤怒其实只是事物脱离自己控制时的一种情绪表达，但是，这不应该是一种对待孩子的情绪，想想恋爱时男女双方对彼此的宠爱和耐心，在外边游玩时碰到熊孩子时的愤怒和无奈，为什么那个时候，你不愤怒，因为你知道恋爱对象是要哄的，其实就算你愤怒了，也不一定有作用；路边的熊孩子，再调皮也是人家的，如果你出离愤怒打了人家孩子，可能会得到对方的群殴或者被叫到派出所渡个三天小长假。而自己家的孩子，在自己身边，没有反抗能力，而且打了就打了，她不会跟你纠缠不休，念念不忘，最重要的一点，她打不过你，而且没有追击性打击。所以，她就成了你的耳巴垫子。

孩子越来越大，学会了谈判讲理，学会了胡搅蛮缠，也许她的本意并不像是大人想象的那样，她只是觉得自己有个理由，自己获得关注，自己获得成就，并没有想去认真的顶撞大人。

当一个人出离愤怒，血管膨胀到失去理智的时候，最好不要去裁定孩子的对和错，更不要想去教育孩子。

但是，大人并不这么想，稍不不如意，就是恶言相向。总感觉孩子这做的不好，那也做的不对，就没有能做好的事情。

一着急，一生气，一上火，就怒从心头起，恶向胆边生。不是恶言相向，就是甩脸子，摆脸色，再者就是开打。

有时候，大人冷漠的眼神和表情，只会给孩子带来恐惧和疑惑，你还不如，当面直接训斥她好呢。当孩子犯错了，大人就直接告诉她，是哪里犯错了，如果你在生气，你也要告诉她，

“你刚才做错的某件事情，我现在还在生气，我现在不想和你说话。”

如果家长啥都不说的在那里一直的冷暴力，给脸色看，别说孩子了，估计热恋中的情商高手也不知道你要啥！

也许我们就不应该盯着孩子的缺点不放。大人，可以犯错，可以放弃，可以偷懒，可以娱乐，唯有孩子不行。

其实很多事情，有时候不加上“打”这个动作，也是可以解决的。

我以前就告诉过娃子，我再打你的时候，你就大声的告诉我，我答应过你不再打你了。

但后来我还是打了…

再后来一次打她的时候，她说，无论你如何凶我，打我，我还是爱你的。

我沉默，无言以对…

当我第三次食言打她的时候，我已经不再和她讲解什么了。她跑去玩，我一个人在书房坐了很久，慢慢平复心情。

第二天我和老婆聊天说，我感觉现在我每打娃子一巴掌，娃子就离我远一点。老婆和我说，娃子现在大了，越来越注意面子了，你以后教育她的时候注意一下，不要伤了她的自尊。

我用五年时间把娃子从一个巴掌大的小人养成小棉袄，不想五年之后，再一巴掌一巴掌的把你打回到一个陌生人。
