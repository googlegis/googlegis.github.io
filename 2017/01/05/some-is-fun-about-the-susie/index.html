##严厉的角色

周末带她去合肥植物园玩，植物园挨着董铺水库，她又酷爱玩水，所以每次去的话，

都会找根树枝，让她在岸边拍水玩，这次也不例外。

等她累了在旁边歇着的时候，我就用树枝用力的在水里拍，结果把水溅到自己身上了，

本来想趁机告诉她，用劲大了就会把水弄到自己衣服上，我刚给她看，还没有开口，

她就严厉的批评到，

“谁让你那么兴奋呢，我都没有你兴奋，下次不准用那么大力气了！”

好厉害的小家伙。
