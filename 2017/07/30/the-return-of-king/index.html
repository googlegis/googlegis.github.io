一个旧号的回归

发表于 2017-07-30 | 分类于 文章 | |

今年二月份，因为一些其它原因，我把公众号转移到了另外一个账号上，平时写些家长里短，育儿故事。

有的时候还会写些影评，书籍和故事。一来满足一下伪文艺青年的创作激情，二来，记录一下孩子的成长点滴，

等老了之后，也算是个念想。

上周，一个没有忍住，写了一篇《为什么要*》，两天之后，公众号被禁，里面的文章和素材全都无法访问了。

有些文章根本就没有存档，也许就这样找不回来了。没有办法，在别人的圈里混，就得遵守别人的规则。当时写

的时候，还特别注意内容和措辞，凡是和政治相关的一点没有敢提及，但是还是被点中。估计申诉无望，还是

重新开始吧。

那些文章虽然是自己写的，部分仍有存档，也许再花点时间整理一下，也能将就的继续发出，但是估计找不到

当时写文章时的感觉和激情了。

从小学时，我就偏爱文学，时常拿高年级兄长的语文书来看，正好，我上小学时能看到初中的书，上初中时能

看到高中的书，等到我上高中时，兄长已经到外地上大学了，没有办法，就去当老师的姥爷和二舅家找，那个

时候，开始接触到外国文学，也曾经被母亲认为不务正业，让二舅来把书都收走了。再后来上了大学后开始

泡图书馆，开始知道了更多的作家和作品，自己偏爱那些农村题材，偏爱近代时间段，也许是特定的历史环境

决定的吧。毕业后，自己有了些散钱，便开始自己购买书籍，大学时看朱苏进的《康熙帝国》，再后来看余华

的《活着》，再到陈忠实的《白鹿原》，这些都是曾经触动心灵的好书。再后来遇到老婆，也是个读书如命的人，

她看的书更杂，一些我不知道的作家她都看，李碧华，严歌苓、辛夷坞，这些我都不知道名字的作家都依次列

在我的书单里。

作为一个文学爱好者，写文章是骨子里流淌着的血液，是一个能够抒发感情的途径，从中学到毕业，从工作到

有孩子，也许那只敲击键盘的手就从来没有停下过。只有写字的时候，内心才能静下来，只有写字的时候，才能

享受那种脑子因为快速运转而分泌多巴胺的快感。每次作文之前，焚香净手，平心静气，把自己关在一个属于

自己的房间里，没有任何杂音和打扰。放开思想的缰绳和束缚，手指在键盘上啪啪的按着，看着一行一行的字

在电脑上跳跃，心中有一种满足和充实感。一篇文章写完，然后校正，保存，上传。当白底黑字的文章最终出现

眼前时，心中会默默的说一声，朋友，我们又见面了。

新号开启了，主要是生活琐事，育儿心得，再也不敢胡咧咧了。总得给那些朋友留个固定的居所。以后的事情再说吧。
