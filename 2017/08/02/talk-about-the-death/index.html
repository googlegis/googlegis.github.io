如何和孩子讨论"死亡"

发表于 2017-08-02 分类于 育儿经 Disqus：

“死亡”这个话题，在儿童教育中貌似和性教育一样，是一个不可触碰的话题，

大多数家长认为，小孩子说死不吉利，一旦孩子说了，就严加职责，但是，大多数家长也只是限制孩子，以后不许说。孩子是不敢说

了，那只是因为她怕你打她，而不是因为她知道，“死”到底是什么意思呢？为什么不让说呢？

最近，娃子放暑假在家，管理的也松泛些，平时我休息时，看电视剧《白鹿原》，里面的生老病死，白嘉轩和鹿子霖

最后都变成白胡子老头了，娃子看着电视问我，

“爸爸，他们是长老了吗？”

“是的。”

“那他们会死吗？”

“会的呀，就像你前面看到的那个人，死了之后就会装进棺材里，然后埋在地里。”

“那你会不会死啊？”

“我也会呀，等我长老了，我也会死。”

说着，说着，娃子哭起来了，

“爸爸，我不让你死，我让你陪着我！”

“那今天晚上我们不讲故事了，等你洗好澡上床睡觉时，爸爸给你讲死是什么好不好？”

“好！”

娃子还特意的跑过去和妈妈交待了一番，

”妈妈，我们快点刷牙洗脸，洗澡后，爸爸给我讲死是什么！”

等娃子，收拾好，上床就开始叫我。

我把手头的东西简单收拾了一下，然后开始给她讲：

如果一个人死了，他就不能说话，不能吃饭，不能走路，也不能和你玩了。死了的人，一般都会有两种处理
方式，一种是被烧成灰，放在一个盒子里，另外一种情况就是把人装在棺材里，埋到地里，然后上面堆起一
个土堆，就是坟。

一般死了的人，都是别人家里的亲人，所以对别人说话的时候，不能随便说谁谁死了，一来别人会恨伤心，
二来会觉得你是一个坏小孩。如果爸爸死了，你是不是很伤心， 如果你说别人家的人死了，别人也会很伤心。
所以千万不能说活着的人死了这样的话，但是有两种是可以说的，一种是动物和植物，比如小猫，小狗，一
种是真正已经死了的人和电视电影里的人。

但是就算真正已经死了的人，还是尽量不要在别人面前提起，因为别人会想起这个人，会伤心。这也是妈妈
平时为什么不让你随便说“死”的原因。

人死的原因，一般也有两种，一种是老死，一种是以外死亡。老死就是人的各个器官使用的时间都太长了，
比如心脏，肌肉，血液都被坏的细菌占领了，没有那么多好的细菌帮你打败它们了，你的心脏就会越来越老，
越来越慢，最后不动了，就不能把氧气和血液带到你身体的各个部分了，如果脑子没有新鲜的血液和氧气，
脑子就不能想东西了， 脑子和心脏不能动了，人体的其它器官手、脚等也就不能动，人也不能呼吸了，
就会死了。所以，平时要保护好自己，尤其是自己的心脏和头，如果你的腿或者手流血了，一般都不会
让人死去，只有流血多了，身体里面的血不能帮你把氧气带到身体的各个部位了才会死去。

娃子问道，“那如果腿出血了，会不会死啊？”

一般也会分两种情况，如果只是腿受伤，流血不算太多，医生能把骨头接好，腿还能动，人不会死，如果骨头
接不好，这个人没有腿了，也是可以活得，比如你看的那个《半边小少年》，他虽然只有一条腿，也是可以活
得，也可以去帮助别人。 另外一种情况，如果腿受伤了，但是一直在流血，又没有人或者医生来救你，就会
流血过多死去。

所以很多时候，如果受伤了，就要及时止血，不能让它一直流。

一般意外死亡都是让你的身体不能提供血液和氧气了，所以才会死亡。比如电死、撞死、淹死。

电死是通过很大力气的电把你身体里面的心脏给停止了，心脏停止了就不能把血液和氧气带到身体的各个部位
了。所以，爸爸妈妈告诉你千万不能摸电。

撞死呢，一般都是汽车有太快的速度，就会有很大的力气，把人的身体给撞伤了，有可能是心脏受伤不能动了
就会死了，也有可能是腿胳膊受伤流血，流血过多就会死了。

淹死呢，是因为无法呼吸道空气，空气中的氧气就不能随着血液流到身体里面，身体的器官缺少氧气了，就会
慢慢死去，然后整个人就会死了。

“那如果一个人在水里，用脚尖站着，但是他的头是在水的上面的，那会不会死啊？”

氧气一般是通过呼吸进入身体的，人体能够呼吸的部位只有鼻孔和嘴巴，如果人的身体是在水里，但是嘴巴或者
鼻孔在水的外面，那就是能够呼吸的，能呼吸就不会死，一般如果人正好在水里的时候，头是在上面的话，就
赶快往岸边走或者游泳，因为越往岸边来水就越浅，获救的机会就越大。

“那会游泳的话，会不会被淹死啊？”

会游泳的话，在水里游泳的时候，能够把头一下一下的露出来，都是在趁露出来的时候呼吸，在水里的时候呢
就赶紧往岸边游，不停的游一下，呼吸一下，这样是不会被淹死的。

“那我现在会游泳就不会被淹死了 ！”

你现在还不算是会游泳，会游泳的话，就是不用救生圈能够自己游很远的距离。爸爸虽然会游一点，但是如果
离岸边太远的话，我就会累的没有力气了，当没有力气不游的时候，人就会沉下去，就会被水淹死了。

被淹死主要还是因为空气变少了，没有了空气人就会死了。比如有很多很高的山，比如喜马拉雅山，青藏高原，
西藏，那些地方，因为位置太高了，上面的空气很少，所以人上去后，喘气就很重，因为呼吸一次要把足够的
氧气给吸进身体里，提供身体的各个器官用。

”那坐飞机会不会死啊？“

坐飞机是不会的，因为坐飞机的时候，飞机是把人包在飞机里面的，飞机里面会提供空气，这样人就能正常呼吸了。

“那人站在高楼上呢？”

地球上面有一圈大气层，大气层里人都是可以正常呼吸的，但是出了大气层，人就要带着氧气。你看电影里的
宇航员到太空中的时候都是穿着宇航服，戴着面罩，里面就有氧气好让他们呼吸，但是如果宇航员进入了宇宙
飞船，宇宙飞船里提供了氧气的话，他们就不用带面罩了。

“爸爸，那人都是要多长时间才会死啊？”

如果是正常老死的话，每个人都不一样，如果一个人锻炼身体，多吃蔬菜，就会有很多好的细菌陪着他打败
坏的细菌，就能活得时间长一点，如果一个人不好好吃饭，整天只吃零食，也不锻炼身体，那就会有很少的
好细菌陪着他，那样的话，坏的小细菌就能很快把他打败，他就会死的早。现在每个人情况不一样的话，有的
人可能60岁就死了，有的人可能要到70岁，而有的人要活到90岁还不会死呢。

“爸爸，我想让你活到最长的时间。”

好啊，那等你长大了，你给爸爸买好多好吃的，那爸爸就能有好多好的小细菌，就有力气打败坏的小细菌，
就能活的时间长了。

“爸爸，那是你先死还是我先死啊？”

一般人的一生会经历出生，长大，结婚，有孩子，变老，然后死去。我现在正处在变老的这个阶段，你呢，
正处在长大的阶段，我比你早生了30多年，就比你大三十多岁，如果是按照正常老死的话，我应该是比你先
死的。

“爸爸，我不想让你死。”

目前爸爸会一直陪在你的身边，等爸爸老了，你也长大了，你也会结婚，也会有自己的孩子，你也会有自己
的家庭，也有有自己的家人陪伴，那时爸爸死了你也不会孤单，也会有人陪你。

“爸爸，我想一直和你在一起。”
