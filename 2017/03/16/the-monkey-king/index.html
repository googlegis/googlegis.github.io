西游1-大圣

发表于 2017-03-16 | 分类于 文章 | |

头戴紫金冠

身披金缕衣

家住花果山

脚踏五彩云

天父地母生

日月孕我身

擎天神针铁

来去筋斗云

湖海纵广阔

难容不平心

戏游蟠桃宴

力战百万兵

如来佛祖至

五行压我身

吃雪饮雨露

盼来取经人

佑护唐三藏

遁入佛家门

万念皆抛去

求修正果身
