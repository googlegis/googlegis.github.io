带孩子的几点想法

发表于 2014-06-05 | 分类于 文章 | |

1. 不要总是盯着缺点

丈母娘从2011年9月份来上海，照顾老婆然后一直照顾小孩，老年人的习惯加上老家的思想，总是感觉处处

不如意，时不时的就莫名的来火上气，总感觉这做的不像样，那也做的不像样，什么都做不好。

有时候，自己脾气上来就给别人脸色看，老婆不高兴，丈母娘不高兴，连着小娃子都要受委屈。

这次独自带娃子去武汉参加同学聚会，真的是感觉到了带小孩的辛苦。

吃饭喝水，拉屎撒尿，闹人睡觉，嘘寒问暖，样样要操心，样样要做到，我自己做来还破费周折，况且还没

有其他人在跟前看着，更没有人会对我检查。即便这样，我还是不能随心应手，更何况一个老年人，照顾自

尚且有困难，还加上一个不怎么听话的娃子，还担心女婿晚上回来看到没有做到的地方会生气。

:-(

我犯了大错了。

前段时间，总是想找个人问问，该如何缓解和丈母娘之间的这种对立的情绪。

这下自然的消除了一部分，虽然有时还是会看到做的不对的生气，但是自我感觉没有以前的那种恨铁不成钢

的气势了，毕竟是老年人了，而且，自己又不是亲儿子，娃子又不是亲孙女，刷锅做饭，洗衣倒水，还不都

为了她闺女，我媳妇吗。

站在媳妇的角度，站在丈母娘的角度，再看看，所有的事情都不值得生气了。

娃子慢慢的长吧。

希望不要丢失了自己的灵性。

2.小孩是要哄的

小娃子哭，小娃子闹，小娃娃天天睡大觉。我的脾气不好，估计和最近的工作压力有关，也和这行的工作

性质有关，喜欢一个人安静的听从自己的安排。电脑听自己的，代码听自己，周围的人都听自己，不允许

有瑕疵，有bug，(话说我是什么星座？)，但是小娃子就不一样了，小娃子有个性，会思考，会讲理，自己

有腿有脚，会自己跑，哪能事事听你的啊。

有时候自己的脾气上来就用强力制止小娃子，小娃子就会用哭来抵抗，自己还总是训她不听话，两败俱伤。

其实你换个方法也许更好，比如买个糖，买个玩具。

如同聚会的时候，如果她不听话，买个糖或者玩具给她，她就会老实下来。

娃子是个吃货，

不要和她较劲，用个糖哄一下，比你教育的效果好。

3.难题

一直都是教育娃子过马路，红灯不能过绿灯能过，娃子也很听话，平时总能遵守规则，时不时带她出去

玩的时候还测试她：这个是什么灯，能不能过？

娃子总能按照红灯停绿灯行的规则回答。

小区后面有个大马路，有拐弯灯，平时过马路时，车辆直行灯变红时，拐弯灯变绿，但是人行道上灯还

是红色的。但是大家根本等不及人行道灯变绿，就直接闯红灯过去了。

有次带她出去玩，回来晚了，当时也没有太注意，正好赶上拐弯灯变绿，就问娃子：现在能不能过？

娃子说：能过！

“那现在不是红灯吗？怎么能过呢？”

“别人都能过了！”

我脑子停顿了一下，别人都能过了，该怎么解释呢？

“别人都不是乖宝宝，上幼儿园回来老师不发小红花，娃子是乖宝宝，上幼儿园老师发小红花。”

出一头汗，总算混过去了。

估计以后还会多次碰到这种情况，别人为什么能插队？别人为什么能受贿？别人为什么能偷钱？

:-(

等她再长大一点慢慢解释吧。

4. 智商

出去玩回来第二天，妈妈有几个口香糖放在桌子上，不知道怎么让娃子看到了，拿到手里不肯松开，

要给她夺下来，还和你讲理：俏俏又不吃，俏俏拿着。

不过那个口香糖的味道确实香，娃子手里掐了一点，放在鼻子上闻闻，放在嘴边舔舔，你一看她，她

就说： 俏俏又不吃。

好，拿着就拿着吧。

妈妈下班了，带她出去接妈妈，外面想下雨，拿着伞，娃子手里还是拿着口香糖。

出了小区，过马路，娃子双手搂着我的脖子，等她回过脸的时候，看她一脸神秘的微笑，我看口香糖上

已经有口水的印子了，问她：你是不是吃口香糖了。

娃子只是笑，过了一会，娃子说：爸爸你打伞吧，下雨了。

其实在树下走，天空还没有开始下雨，

“没有下雨啊”

“你看，都滴到俏俏的脸上了”

“在哪里？” 我问，

“你看你看，”

她这么一说，我还真觉得有点下了，我当时是一个胳膊抱着她，一个手拿着伞，还得用另外一个手帮着

撑开伞，就在我撑开伞的那一个瞬间，我看见娃子的手，迅速的把口香糖放到嘴里，然后快速的闭上嘴。

我连忙从她嘴里把口香糖扣出来，她自然是不愿意，哭得稀里哗啦，

我想既然哭了，索性把手里的都扔了吧。她双手攥的紧紧的，哭得昏天暗地。她去打疫苗时都没有哭。

小娃子，这个东西你不能吃。

等她稍微缓了点劲，告诉她，“有些东西可以吃，有些东西不能吃，棒棒糖可以吃，口香糖不能吃。”

最后买了个棒棒糖哄她，才停止啜泣。

晚上的时候，在想，小娃子告诉我下雨了，让我去打伞，是不是早就准备好了要吃口香糖呢，

她是不是知道我一直盯着她的话，她就不能吃到嘴里，必须要趁我不注意快速的吃掉呢。

那她是怎么想起让我去打伞的这个机会的呢？

这么说来整个事情都是小娃子的预谋了。

两岁半的小娃子就能这么思考(也许是本能)，比你爹强。
