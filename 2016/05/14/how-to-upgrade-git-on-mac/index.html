如何升级Mac上的git

发表于 2016-05-14 分类于 Mac

首先 需要检测仪安装的git的版本，貌似现在 mac 下面是默认安装 git 的说。

我不知道我的电脑上是什么时候安装的，但是可以直接用。

在 命令行中输入：

下面开始升级：

首先，从官网下载升级包，

git-2.8.1-intel-universal-mavericks.dmg

加载后，打开 readme，里面有个卸载的命令，先运行一下。

双击安装包

git-2.8.1-intel-universal-mavericks.pkg

一路Next。

默认安装 Git 到 /usr/local/git (需要输入mac密码赋予权限)

然后完成安装。

查看 git 版本

这样就完成了 git 的生机。

如果命令行中仍然显示旧的版本号，不要着急，只是程序的路径没有添加到环境变量中。

打开命令行工具：

jays-MacBook-Pro:~ jay$ vim ~/.bash_profile

把下面这行添加到 .bash_profile 文件的后面，

export PATH=/usr/local/git/bin:$PATH

保存文件，退出编辑！

再验证版本，就会看到最新的升级版本号。
