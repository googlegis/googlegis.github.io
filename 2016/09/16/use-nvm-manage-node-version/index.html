使用 nvm 来管理 node 的版本

发表于 2016-09-16 | 分类于 Mac | |

使用 nodejs 的人都知道，node 是个版本狂魔，你不知道啥时候，它就发布了
一个新版本。

不过从 nodejs 的发展历程来看，也真是难为它了。

不管怎么说，多版本共存的安装和管理工具确实是必不可少的。

幸运的是 nvm 就是这一工具的颜值担当。

安装nvm

可能是 ~/.bash_profile 或者 ~/.profile 或者 ~/.bashrc 文件中。
