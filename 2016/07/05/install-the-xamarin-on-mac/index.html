在 Mac 上安装 Xamarin

发表于 2016-07-05 分类于 Xamarin

最近因为工作的原因，要把已有的 .Net Web 项目进行跨平台开发研究，项目部署

在 CentOS 上，正好有机会研究一下 Xamarin 的开发，所以决定把这一系列

的研究过程都记录下来，备忘和分享！

.Net 的跨平台开发，现在无非是两种方式， 一种是 Mono SDK, 一种是

Asp.Net Core. 但是 Asp.Net Core 还是太新，只能作为对比，还不敢拿

来直接作为开发使用。所以，默认使用 Mono。

Mono 是一个跨平台的 .Net 解决方案，最后被微软收购并进行维护和支持。

应该说还是有一定的群众基础的。

在 Linux 系统下推荐的是 MonoDeveloper

http://www.monodevelop.com

在 Windows 和 Mac 系统下推荐的是 Xamarin Studio。

http://www.monodevelop.com/download/

从 Mono-Develop 的官方页面下载的版本是 5.6

但是从 Xamarin 的官方页面下载的是 6.0

https://store.xamarin.com/account/my/subscription/downloads

如果只是开发 Web 页面，只下载 Xamarin Studio 的安装包即可。

然后直接双击安装， 如果没有安装 Mono SDK，会提示要先安装 SDK。当然也可以

从网站先下载安装 Mono SDK，安装再安装 Xamarin Studio。

http://www.mono-project.com/download/

打开 Xamarin Studio，新建 Asp.Net Mvc 项目进行测试。

至此，说明 Xamarin Studio 安装成功！
